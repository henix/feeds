官网：https://blackarch.org/tools.html

安装（在ArchLinux之上安装）

它充当非官方的用户存储库。

＃以 root 身份运行 https://blackarch.org/strap.sh 并按照说明进行操作。
$ curl -O https://blackarch.org/strap.sh ＃SHA1的总和应该匹配：9f770789df3b7803105e5fbc19212889674cd503trap.sh
$ sha1sumtrap.sh ＃设置执行
$ chmod + xtrap.sh ＃运行trap.sh
$ sudo ./strap.sh

现在可以从 blackarch 存储库安装工具。

＃要列出所有可用工具，请运行
$ sudo pacman -Sgg | grep blackarch | 切-d''-f2 | 排序-u＃要安装所有工具，请运行
$ sudo pacman -S blackarch＃要安装一类工具，请运行
$ sudo pacman -S blackarch- <类别>＃要查看blackarch类别，请运行
$ sudo pacman -Sg | grep blackarch
＃注意-安装blackarch工具时，可能有必要覆盖某些软件包。如果
＃遇到“failed to commit transaction”错误，请使用--needed和--overwrite开关
＃例如：
sudo pacman -Syyu --needed blackarch --overwrite='*'

作为替代安装方法的一部分，您可以从源代码构建blackarch软件包。可以在github上找到PKGBUILD 。

https://github.com/BlackArch/blackarch

＃首先，您必须安装 blackman。如果您的计算机上安装了 BlackArch 软件包存储库，则
＃您可以像这样安装 blackman：
$ sudo pacman -S布莱克曼＃下载，编译和安装软件包：
$ sudo blackman -i <程序包>＃下载，编译和安装整个类别
$ sudo blackman -g <组>＃下载，编译和安装所有BlackArch工具
$ sudo blackman -a＃列出blackarch类别
$ blackman -l＃列出类别工具
$ blackman -p <类别>

ISO上安装

可以使用 Live 或 Netinstall 介质安装 BlackArch Linux（程序包和环境）。

＃安装 blackarch-install-scripts 软件包
$ sudo pacman -S blackarch-install-scripts＃现在，您可以运行并按照说明进行操作
$ sudo blackarch 安装
