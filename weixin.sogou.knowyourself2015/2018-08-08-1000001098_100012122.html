戳穿系心理，不是治愈系心理。人生尽是幻象，你要敢于戳穿。


今天，KY开启秋季实习生招聘：


我们希望按照KY信奉并一直在努力践行的认真、诚实、肯干的工作准则，结合我们赋权、平等、自由、多元的价值观，为大家营造一个能够在实习工作中找到真实自我、并参与创造社会价值的创业环境。


希望大家能在KY倡导并践行的工作氛围中，体会到从学生到职场人的转变：感知自己肩上的责任，体会自己的行为对他人的影响。


我们希望邀请更多和我们一样，受过（某方面的）专业的训练、喜欢创新和不断迎接挑战、有意愿加入KY的小伙伴，成为我们并肩的战友。


实习地点：上海市长宁区，地铁直达写字楼


适宜人群：

1. 2019届毕业生，本科、硕士均可；

2. 可保证每周（周一至周五）全职工作最少3天，实习期三个月及以上 。


表现优秀者有转正机会哦，来投简历吧！

以下是在招实习岗位：


【 KY线下实习 】


- 内容中心 -


· 专业研发实习生·


工作内容：

1. 进行心理测试的研发、试测及修订，基于对相关前沿研究及大众心理/行为的洞察和理解；

2. 负责测试相关文案写作，测试结果报告的数据可视化等

3. 利用相关数据进行用户解读、人群细分等工作，并将数据进行可视化，撰写研究报告，协助公司更深入了解用户需求。


任职要求:

1. 本科及以上学历，心理学、社会学或社会工作专业；

2. 有专业心理量表的研发经验，具备学术研究与写作能力, 具备优秀英文阅读与中文写作能力；

3. 逻辑思维强；

4. 熟练使用excel/SPSS/R等数据分析软件，会使用Atlas ti或Nvivo软件加分

5. 快速学习能力强，

6. 责任心强；

7. 开放性思维，不局限于现有内容方向与形式，愿意打破常规。



· 内容实习生·


工作内容：

1. 参与KY内容在知乎、今日头条、微博、豆瓣等平台上的改编创作和分发；

2. 参与知乎问答、悟空问答、感情问答等不同平台和不同主题内容有关的问题答复；

3. 参与心理专业科普内容的收集整理，比如实验、研究等；

4. 参与人文社科美学领域的素材收集，比如经典文选、名画名曲；

5. 参与热点事件的响应和创作。


任职要求：

1. 熟悉KnowYourself的文章；

2. 拥有本科及以上学历，其中心理学、社会学或社会工作专业优先考虑；

3. 平日里有使用知乎、今日头条、豆瓣、微博等平台的习惯，对互联网新近发生的事情心里有数，网感好；

4. 具备良好的中文写作能力，逻辑性好，英文阅读能力较好；行文清晰准确、简洁；

5. 耐心，仔细，认真，负责，抗压能力强，有团队合作精神；会摄影，会作图，有策划活动的思维是加分项。



- 市场中心 -


· 市场中心实习生·


工作内容：

1. 协助部门内对外市场合作对接，公司内外资源的沟通，把控项目进度；

2. 参与到市场活动的创意策划、落地执行和宣传推广中；


任职要求：

1. 985、211高校在读生，大四及研究生三年级优先，一周至少全职工作4天；

2. 快速学习能力强，有网感，脑洞大，责任心强，沟通力强；

3. 热爱心理学、对自媒体行业有了解。



· 音频策划实习生·


工作内容： 

1.负责广播音频自制类节目，挑选选题，策划节目形式，文案改写，跟进外包制作，以及剪辑和上线推广节目，能够把握一档节目的全案策划运营；

2.逻辑思维清晰，文案能力强，选题上手快； 

3.对互联网用户的需求有敏感度； 

4.想法多、团队合作意识强、沟通能力好，能承受一定的工作压力，不拖延；

   

任职要求：

1.大学本科及以上学历； 

2.熟练使用Audition、Premiere等软件；

3.新闻、广播电视编导及录音、播音主持等专业方向优先； 

4.有1年以上在广播、电视台、视频网站、制作公司工作经历者优先；

5.思维活跃、脑洞大、有阅读储备和良好的学习能力者优先。



· 视频实习生·


工作内容：

1. 负责短视频前期策划以及文案脚本等工作；

2. 负责视频的拍摄、后期剪辑和素材搜集整理；

3. 能进行简单的后期处理；

4. 负责视频栏目运营相关工作，包括内容上传，合作认证等；

5. 视频竞品分析，以及每周数据整理分析。


任职要求：

1. 熟练使用FCP、PR等剪辑软件，熟悉AE、AU等软件优先；

2. 掌握摄像技术和应用技巧者优先；

3. 熟悉流行短视频应用、二次元文化优先；

4. 做事认真负责，善于学习，沟通能力强，善于协调工作。



- 运营中心实习生 -


工作内容：

1.群运营：协助维护运营KY各类粉丝群；

2.课程跟进：协助开课，维持课堂、学习群秩序，整理课件等学习资料；

3.微信运营：图文排版、创新形式研发、关键词设置等；

4.运营互动活动创意策划：监测其他平台优秀活动，策划我们的活动；

5.客服工作：协助回复用户问题，安抚用户情绪；

6.数据统计：熟练使用Excel表进行数据统计；

7.协助联系咨询师进行相关合作。



任职要求：

1.大三或大四在校学生(或课业压力较小的研究生)心理学及社科背景优先；

2. 可保证每周工作3-5天，工作时间10:00-19:00，实习期在三个月以上 ；

3.脑洞能力强，有创意有想法，并且有把事情执行出来的动力和能力；

4.工作认真仔细负责，有耐心沉得住气，能从琐碎事情中体会工作乐趣；

5.会微信排版(及各类排版工具)、PS基本操作，会HTML5语言或者简单编程者加分；

6.有较强的沟通能力，英语较好。



- 设计部 -


· 视觉设计实习生·


工作内容：

1. 线上运营设计：banner、活动页等。

2. 企业公号封面设计。

3. 其他日常视觉设计需求。


任职要求：

1. 设计或艺术类相关专业。

2. 一周到岗5天。

3. 熟练操作AI、PS等设计软件。

4. 有插画能力或者字体设计能力的优先考虑



· UI设计实习生·


工作内容：

1.小程序和APP的设计和优化。

2.一些运营设计需求：banner、活动页等。

3.其他视觉设计需求。


任职要求：

1. 设计或艺术类相关专业。

2. 一周到岗5天。

3. 熟练操作AI、PS、Sketch等设计软件。

4. 有手绘能力的优先考虑。



- 总裁办 -


· 行政实习生·


工作职责： 
1.负责办公物品申购、各种用品的记录管理及分发工作； 
2.负责公司快递费，水费等各项费用的缴纳结算； 
3.负责部门财务发票的整理以及报销工作； 
4.负责办公环境、固定资产的维护及日常管理； 
5.协助组织公司内部各项定期和不定期集体活动； 
6.负责员工每月考勤核算，社保、公积金，及相关税务的缴纳业务； 
7.支持各部门的同事做好其他相关工作。


任职要求： 
1.本科及以上学历，专业不限； 
2.工作认真、负责、踏实，能够及时响应且灵活处理好各项事务。 
3.具有良好的的沟通、组织、协调能力和团队协作能力。



· 人事实习生·


工作内容：

1.协助考勤管理，员工加班与假期登记，每月打卡记录处理 ；

2.协助面试接待，办理员工入职离职续签手续和前期材料准备； 

3.负责实习生招聘、入职、离职、考勤相关事宜 ；

4.资料收集、归档、整理、发放 ；

5.部门主管及同事安排的其他事务 。


任职要求：

1. 大三或大四在校学生，大三学生优先考虑；

2. 可保证每周工作最少4天，工作时间10:00-19:00，实习期在三个月以上 

3. 工作认真仔细负责，具备Office基本技能，PS熟练使用，有较强的沟通能力，英语较好。



- 产品中心实习生 -


职位描述：

1. 负责市场需求收集、参与用户调研，做好竞品分析；

2. 协助产品经理，进行产品规划、业务流程设计、功能设计及相关文档撰写； 

3. 协助跟进运营需求，在产品负责人的带领下设计产品迭代需求；

4. 与研发、运营等团队进行沟通确保对产品的理解，协助推进开发，跟进测试和上线。


任职要求:

1. 2019届毕业生（本科、硕士均可）；

2. 能够使用Axure/Visio软件；

3. 有良好的逻辑思维与分析能力，学习理解能力强；

4. 具有良好的交流沟通能力，敏感的数据意识；

5. 具备数据收集分析，文档撰写能力。



- 销售中心实习生 -


工作内容：

1.负责广告项目的执行，包括客户对接、内部协调、流程管控、供应商管理等；

2.负责搜集整理行业及竞品动态，并制作分析报告；

3.负责支持销售中心其他工作。


任职要求：

1.重点大学本科及以上学历，一周工作至少4天，可工作时长3个月起；

2.工作踏实，执行能力强，能多线程管理工作，条理清晰不出错；

3.熟练掌握Microsoft Office、PS等办公软件，能高效制作各类文档；

4.有较强的责任心，具备快速学习成长能力；

5.个性开朗，善解人意，擅长沟通。

- KY线上专业运营志愿者 -


工作内容：

1. 解答用户专业类疑问（“专业问题”：区别于事务性回复、是涉及心理问题的问题）；

4. 总结回复情况并对KY提供内容选题上的建设性反馈。


任职要求：

3. 有较强执行力，具有较好的沟通协调能力和服务意识，耐心细致，具有良好的职业操守；

4. 熟悉KY的文章，跟进了解KY各活动；

5. 具备良好的中文沟通及写作能力，能准确清晰表达内容。


工作时间：

线上志愿者按每周轮班制进行工作
