KnowYourself

knowyourself2015

泛心理科普与服务，美好生活从了解自我开始。


KnowYourself不仅仅是个公众号，它是一家心理学领域的互联网创业公司。

 

我们的使命，是让每个人拥有幸福的能力。

 

旗下拥有泛心理领域用户质量最高，流量最大的媒体品牌KnowYourself，在全网已聚集超过650+万用户，建立起包括心理测评、知识付费、轻咨询、互助社群等在内的立体业务矩阵。

 

我们想在社会上提倡一种新的健康观，从传统的生理健康，转向身心健康。我们希望让追求心理健康，像健身一样成为流行的生活方式。

 

KnowYourself的企业文化关键词是：创造力、团队意识、自我驱动和真诚。我们希望你具备这四点特质。

 

在这里你将获得:

心理研发部


· 研发专员 ·（心理知识产品方向）

（全职/实习）


工作内容：

1. 针对大众感兴趣的泛心理相关话题，研发相关的心理类“产品”，包括但不限于知识付费产品；

2. 与行业专家进行沟通，主导或协助搭建课程框架；

3. 关注竞品与市场动态，持续优化内容与用户体验。


任职要求：

1. 本科以上学历，重点大学心理学、社会工作等相关专业；

2. 对心理学充满热情，专业基础扎实，也有志用心理学相关知识打造出创新性的“产品”；

3. 拥有创造力、想象力，逻辑清晰、熟悉互联网；

4. 有极强的学习能力，有数据分析能力加分；

5. 热爱挑战，不拘泥于现有的方向与形式，敢于打破常规

6. 责任心强，擅长团队协作，项目管理能力加分。



· 研发专员 ·（亲子方向）

（全职/实习）


工作内容：

1. 协助统筹推进KY公众号矩阵-亲子号的运营；

2. 搜集相关话题热点，参与公众号的选题、创作；

3. 负责公众号的内容运营；

4. 关注竞品与市场动态，持续优化内容与用户体验；

5. 部⻔的其他日常工作。

 

任职要求: 

1. 对亲子、养育话题充满热情；

2. 专业基础扎实，乐于将亲子养育方面的专业心理学知识以深入浅出的形式推广给大众；

3. 本科以上学历，重点大学社会工作、教育学、心理学等专业优先，对发展心理学具备一定的基础知识加分；

4. 熟悉互联网，网感强，具备互动运营的能力，有过相关的新媒体经验加分；

5. 具有一定的项目管理、统筹能力，具有创造力、想象力，逻辑清晰；

6. 具备优秀的文字功底，有过撰写稿件的经历加分；

7. 有能力阅读和听写英文材料 ；

8. 热爱挑战，不拘泥于现有的方向与形式，敢于打破常规；

9. 责任心强，擅⻓团队协作，学习能力强，能承受较强压力与较快节奏。



商业化中心


· 整合营销经理 ·


工作内容：

1. 根据客户需求，负责整合营销方案策划，提案，并协助销售促成合作；

2. 整合内外部资源，在团队协助下推进营销方案的执行和管理；

3. 实时监测市场动向，分析总结，研发广告新形式。


任职要求：

1. 重点大学本科及以上学历，2年及以上广告/市场相关工作经验；

2. 具备优秀的创意思维和市场敏感度，策划能力强；

3. 个性开朗，有良好的沟通协调能力；

4. 踏实认真，具备团队合作精神、责任心、快速学习能力；

5. 能熟练使用办公软件制作提案，英语流利。



产品技术中心


· 数据分析 ·


工作内容：

1. 熟悉互联网常用数据分析指标，掌握产品运营数据的监控和分析方法；

2. 负责分析和解释产品试验，市场活动等的结果，为产品改进，推广试验等提供数据支持； 

3. 分析用户特征、行为路径、转化率、活动效果等运营核心数据，提供数据反馈和建议 ； 

4. 对运营数据进行挖掘和分析，对运营策略的制定和优化予以数据支持 ； 

5. 具有独立承担复杂分析任务的能力，支持各方不定期的相关数据需求。


任职要求:

1. 计算机、统计、数学、信息技术等专业本科及以上学历；

2. 至少2年以上互联网数据分析或大数据相关岗位工作经验；

3. 熟悉SQL、Oracle、MySQL，熟悉数据相关工具； 

4. 熟悉python/R等脚本语言开发者优先；

5. 有产品思维，对商业和业务逻辑敏感；

6. 具备较强的创新思维和逻辑分析能力，良好的书面和口头表达能力。



· 用户运营 ·（用户增长方向）


工作内容：

1. 负责公司互联网产品的用户增长，特别是微信生态下裂变增长建设：拉新、促活、留存、裂变扩群；

2. 制定增长目标并分解计划，灵活运用各种推广渠道及用户增长方式，实现用户规模化增长；

3. 以用户增长为目标，不断探索新颖有效的用户拉新手段，提升整体拉新效果；

4. 配合公司节奏，有针对性地独立策划线上活动，跟踪分析各环节漏斗数据，持续优化用户增长及推广方案；

5. 有一定的数据分析能力，能对产品和市场数据进行分析，并以此为依据不断推进增长方案的改进。


任职要求：

1. 本科及以上学历，2年及以上互联网产品用户增长或市场营销相关工作经验； 

2. 具有良好的结构化思考能力，逻辑思维清晰，对数据敏感；

3. 具有较强的文案策划、营销推广能力。有内容营销、H5推广、小程序裂变等成功案例者加分；

4. 有从0-1负责或参与用户增长项目经验者加分；

5. 有产品敏感性，平时喜欢体验各种产品，熟悉使用Axure、Mindmanager、Visio等工具；

6. 有团队意识，同时具备良好的项目协调、推进能力。



媒体部


· 内容运营 ·

（全职/实习）


工作内容：

1. 参与微信公众号的相关运营工作，包括推文排版，评论留言维护，公众号页面维护，内容搜索更新等；

2. 时刻关注市面上各种内容运营形式和玩法，并配合作者们将每一篇稿件作为产品去打磨、包装和运营（e.g. 添加插件）；

3. 通过创新不同的方法提升内容的获新能力(e.g. 转发量)，以及用户的活跃(e.g.打开率、点赞、评论、留言)；

4. 参与KY条漫项目的内容创意和策划；

5. 运营互动活动创意策划：监测其他平台优秀活动，参与策划KY的运营活动。


任职要求：

1. 熟悉微信排版(及各类排版工具)、PS基本操作；

2. 本科及以上学历，拥有心理学相关知识加分；

3. 快速学习能力强、责任心强，认真细心，有联动意识；

4. 熟悉互联网各大内容平台，有微信公众号运营经验优先；

5. 开放性思维，不局限于已有框架，有创意、脑洞大。



心理咨询部


· 咨询师运营 ·

（全职/实习）


工作内容：

1. 心理咨询预约平台的拉新审核入驻及平台使用问题答疑；

2. 咨询师社区运营，咨询师合作及关系维护；

3. 组织咨询师内容生产；

4. 线下咨询空间相关工作。


任职要求：

1. 本科以上学历，重点大学应用心理学、心理学、社会学、社会工作等相关专业；

2. 对心理咨询师群体有了解，拥有咨询机构助理、运营经验优先；

3. 有良好的心理学专业理论基础，了解国内的心理学流派和分布，对行业内的老师和咨询师有一定的资源人脉积累；

4. 沟通能力强，具备良好的文案撰写能力；

5. 逻辑思维能力强，有执行力。



市场品牌部


· 市场品牌经理 ·


工作内容：

1. 拓展与公司品牌调性相符的异业合作伙伴，挖掘合作点，策划联合营销；

2. 整合公司内外的可利用资源，扩大联合营销活动的影响力，有效转化潜在用户；

3. 搜集活动数据，分析不同渠道的转化效果，为品牌联盟的建立打磨方法论；

4. 处理公司日常公关事务，通过撰写公关稿件、参与行业论坛等方式提升公司在业内的品牌形象，阐述公司产品理念及价值观；

5. 负责处理公司及管理团队的采访需求对接及内容输出。


任职要求：

1. 本科以上学历，2年以上工作经验；

2. 完整策划并执行过互联网品牌间的异业合作、或不同领域的跨界营销，能总结出自己的方法；

3. 沟通能力强，善于商务谈判，且有创意策划能力；

4. 知名互联网公司市场合作岗从业经历优先。



总裁办


· 人力资源经理 ·


工作内容：

1. 根据用人部门需求制定招聘计划，独立负责落地执行招聘相关工作；

2. 负责公司员工入离职、合同签订归档、考勤及工资核算等员工关系工作； 

3. 负责统筹及执行薪酬绩效福利制度，组织开展员工培训等；

4. 统筹人力资源各个模块，总结人事工作中存在的问题，完善人事制度及流程。


任职要求：

1. 五年左右人力资源及相关工作经验，有人力资源全模块工作经验，其中在招聘和薪酬两大模块有一年以上工作经验。

2. 熟悉互联网相关行业优先； 

3. 执行力强，学习能力强，优秀的沟通协调能力和分析判断力； 

4. 良好的职业道德及团队合作意识，良好的亲和力。


* 工作地点：上海市长宁区
