原文

为什么要编制

和股票不一样，一个期货品种往往同时存在多个期货合约，每个合约都有高开低收量，如何追踪期货品种的行情变化，是期货研究人员第一步需要解决的。目前提供期货归类和数据的机构很多（连祥斌和桂琪琪，2018），他们提供的指数大多是价格指数，用来看盘看走势还行，用来做量化研究多多少少会有问题。

在做研究时，一个常被使用的价格数据是主力合约指数。例如tb的888合约，就是简单地将成交量最大的合约作为主力，然后拼接主力数据得到主力合约指数。利用主力价格指数得到的收益不完全是真实收益。例如，就一个长期远期升水（contango）的商品而言，如果将其主力合约的历史数据连起来，我们发现它的K线图是震荡上扬的（换月时有很多跳空）。如果投资者对该商品进行一个长期的买入操作，其结果却并不一定能盈利。这是因为该商品的远期合约价格往往较高，甚至比前一个主力合约的同期价格高出很多，这是导致主力合约价格走势震荡上扬的原因，但是投资者是无法从这个价差中获取收益的。因为期货不像股票，投资者不能长期地持有某一个月份的商品合约，当该月份的商品进入交割月时，投资者需要换仓，而移仓所发生的价差是要相应扣除的。

对主力合约指数的一个替代方案是合约加权指数，例如tb的000合约，它将t日所有合约的价格按照volume和openint以一定的方式进行加权得来。加权合约指数虽然解决了主力合约指数跳空的问题，但它还是不能反映持有商品期货合约的真实收益。首先，它是所有合约的加权，交易时持有所有合约管理难度较大，不现实；其次，不同合约持仓和成交不断变化，权重也不稳定，使得合约指数难以实盘复制。

既然常用的主力合约指数和加权合约指数都有一些瑕疵，那么我们应该使用怎样的数据才能反映投资者的真实收益水平呢？一个有效的方案是收益率指数。

单个商品指数主要是通过剔除换月时因前后合约价格不同而产生的价差所形成的指数，从而达到跟踪和反映投资者在商品期货投资中真实收益。因为指数的计算主要是以收益率来计算的，因此，也将其称为收益率指数。显然，商品收益率指数的最大优势在于体现了投资者的真实收益。

在国内，南华收益率编制的收益率指数质量较高，我们的编制方案和南华相似。

方案回顾

通常来说，构造期货价格指数会把主力合约价格序列拼接起来，再进行一定调整。为了研究期货时更准确地做回测和技术分析，一般我们需要调整后的序列在换月时不出现跳空，能平稳换月，也需要该价格序列能反映真实收益。 下面总结几种常见方式（Masteika et al., 2012）：

  • 向前调整(Forward-adjusted)
    即向未来方向调整。每次换月就将未来方向的合约价格序列平移至消除跳空，相当于未来方向的合约价格序列都加上跳空。这种方法的优点是不用重复计算历史数据；缺点是未来可能出现负值的价格，而且由于是平移变化，该序列不能反映真实收益率。
  • 向后调整(Back-adjusted)
    即向历史方向调整。每次换月就将历史方向的合约价格序列平移至消除跳空，相当于历史方向的合约价格序列都加上跳空。这种方法的有优点是可以反应当前主力合约真实价格；缺点是需要不断重新计算历史数据，且历史数据中可能出现负值的价格，同样地，也不能反映真实收益率。
  • 按比例调整(Proportionally adjusted)
    每次换月按比例调整价格序列以消除跳空。通过以上1.和2.的优劣势分析，为了避免不断重新计算历史数据，选择用向前的按比例调整方法。具体方式如下：
    假设需要连接合约1（到期时间点为T1）和合约2（到期时间点为T2，T2>T1），换月时间为t，换月时合约1的价格为Ft,T1，合约2的价格为Ft,T2。调整合约2的价格序列，使得合约2在t时刻的价格为Ft,T1。因此调整比例为Ft,T1/Ft,T2，合约2的价格序列上所有价格都乘以该比例。
    换月前的序列没有变化，符合真实收益率。一般情况下换月时是在收盘价上卖合约1，同时买合约2，这样换月当天的收益就是合约1的收益，调整后的序列在换月时价格就是合约1的价格，因此调整后的序列符合真实收益率。换月后的序列是同比率变化，跟真实的合约的收益率相同。
    综上，向前的按比例调整方法可以反映真实收益率，不会出现负值的价格，也不用重复更新历史数据。
  • 永久序列(Perpetual series)
    这种方式在换月时根据成交量或者持仓量加权，混合前后两个合约价格，分多日换月，这样可以平滑价格序列。本质上没有解决反映真实收益率的问题，但分多日调仓的方式适用于构造收益率指数。

我们的方案

商品期货和国债期货

1.计算方法

合约的选择
国内各商品在各年度的交易的主力合约有所不同，不能采用固定换月的方式。我们选择商品期货所有上市合约中持仓量最大的合约，即主力合约，进行动态换月。国债期货在交易中也采用类似方式确定换月时间，因此也可以使用这种方法计算指数。

换月策略
当一个主力合约转换到另一个合约时（一般判断条件是下一个合约连续3天的持仓量最大），分5天将持仓头寸平均转移到下一个合约。

具体计算
具体计算方法如下：

首先，定义各个变量

① 初始时
index(i,0) = 1000
hold(i,0) = index(i,0) / price(i,m,0) = 1000 / price(i,m,0)

② 不涉及换月的交易日t
hold(i,t) = index(i,t-1) / price(i,m,t-1)
index(i,t) = hold(i,t) * price(i,m,t)

③ 换月时
如果满足换月条件（一般条件为，连续三天满足int(i,n,t)>int(i,m,t)，具体规定下面有详细说明），则需要对该指数进行展期，展期过程共分5天完成，且采取平均转移头寸的方式。

第一天（满足换月条件的当日日末）
hold(i,t) = index(i,t-1) / price(i,m,t-1) -------- 表示当日换月前合约m的持仓量，在该换月期间数值不变。
hold(i,m,t) = 0.8 * hold(i,t)
hold(i,n,t) = 0.2 * hold(i,t) * price(i,m,t) / price(i,n,t)
index(i,t) = hold(i,m,t) * price(i,m,t) + hold(i,n,t) * price(i,n,t) = hold(i,t) * price(i,m,t)

第二天
hold(i,m,t+1) = 0.6 * hold(i,t)
hold(i,n,t+1) = 0.2 * hold(i,t) * price(i,m,t+1) / price(i,n,t+1) + hold(i,n,t)
index(i,t+1) = hold(i,m,t+1) * price(i,m,t+1) + hold(i,n,t+1) * price(i,n,t+1)

第三天
hold(i,m,t+2) = 0.4 * hold(i,t)
hold(i,n,t+2) = 0.2 * hold(i,t) * price(i,m,t+2) / price(i,n,t+2) + hold(i,n,t+1)
index(i,t+2) = hold(i,m,t+2) * price(i,m,t+2) + hold(i,n,t+2) * price(i,n,t+2)

第四天
hold(i,m,t+3) = 0.2 * hold(i,t)
hold(i,n,t+3) = 0.2 * hold(i,t) * price(i,m,t+3) / price(i,n,t+3) + hold(i,n,t+2)
index(i,t+3) = hold(i,m,t+3) * price(i,m,t+3) + hold(i,n,t+3) * price(i,n,t+3)

第五天
hold(i,m,t+4) = 0 * hold(i,t) = 0
hold(i,n,t+4) = 0.2 * hold(i,t) * price(i,m,t+4) / price(i,n,t+4) + hold(i,n,t+3)
index(i,t+4) = hold(i,m,t+4) * price(i,m,t+4) + hold(i,n,t+4) * price(i,n,t+4)
换月结束。持有的合约已从合约m换为合约n。

2.关于展期的其他规定

  • 展期方向规定
    如果出现主力合约往后变动的情况，则不展期；保证展期只能往前顺移，不能往后。
  • 一般展期中持仓量最大情形规定
    若存在当前主力合约之后有多个合约持仓量大于当前主力合约，则在满足展期条件的那一天（持仓量连续三天大于当前主力合约）选取持仓量最大的合约展期。
  • 强制展期规定
    若当前主力合约在距离交割月前一个月的倒数第5个交易日依旧是持仓量最大的合约，则在从此日开始，寻找当前主力合约以后的合约中持仓量最大的合约向前展期，同样分5个交易日移仓完毕，其中规定当前主力合约的远月合约中持仓量最大的合约为新的主力合约。
    例外：燃料油会提前一个月进入交割月，在计算燃料油指数时，强制展期规定需要调整。

股指期货

股指期货的换月规则相对简单，一般都在最后交割日（也就是最后交易日）的前几天分批换月。具体方案为，从每个月第三个星期的星期一起，分三天完成换月，换到下月合约。

应用场景

期货收益率指数应用范围极其广泛，其设计和计算只是期货研究的第一步。包括因子挖掘、组合构造、风险管理和归因分析，都会出现期货收益率指数的身影。这里举例探讨期货指数的两个应用：平均相关系数和因子指数。

平均相关系数

商品之间的平均相关系数是影响CTA策略的重要因素（Nick Baltas, 2015 ），利用量华期货指数，可以跟踪商品之间的平均相关系数。计算活跃商品滚动6个月平均相关系数如下，可以看到商品之间平均相关系数持续降低，对CTA策略组合分散极其有利。

在对策略进行历史回溯时，用期货指数使得结果更可信，更加逼近真实交易业绩。量华CTA策略主要包括两大因子：趋势跟踪和商品对冲（量华，2017）。为了追踪两个因子的表现，我们利用期货指数编制了策略指数。
