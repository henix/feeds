原文

CoinUp是目前首款集交易所、行情、资讯、对冲搬砖等操作简单易用的一款量化搬砖软件。

授权API：

首先在我的账户授权所需要用到的交易所，要使用自动对冲功能，首先必须设置交易所的API。

什么是API呢？
1、应用程序接口（Application Program Interface）的简称。顾名思义，也就是给程序用的接口。
2、API是一组定义、程序及协议的集合，通过 API 接口实现计算机软件之间的相互通信。
3、API 的一个主要功能是提供通用功能集。程序员通过调用 API 函数对应用程序进行开发

以OKEX为例，首先从右上角头像进入安全设置，选择"新建API"



然后输入备注名新建一个API Key。



这里建议大家选择选择不开放提币，尽管这个这个软件我进行反爬，安全性还不错，但以防万一，万一被黑客攻击呢，还是选择不开放提币为妥。

接下来打开CoinUp，登陆后点击“我的账户”。



点击授权交易平台，输入刚刚创建的APIkey和secret。



自动对冲设置

首先打开CoinUp，点击“对冲”进入对冲页面，然后选择“策略设置”，进行对冲策略参数设定，就可以自动对冲了。

在策略设置页面选择需要对冲的币种EOS，基准货币ETH（ETH、BTC、USDT）以及交易市场选择huobi和Binance(币安)。在对冲策略列表里可以配置最小收益和每次交易量。建议把每次最多交易量调小，实行小额多次交易，降低失败风险。



持仓配置

根据自己的账户情况以及对冲需要配置持仓，



对冲策略设置好以后，点击开始按钮，程序就开始执行了。程序会通过API拉取报价信息，监控价差，一旦满足你设定的利润率，就会执行对冲交易。在程序界面的下方可以看到所有的交易。
