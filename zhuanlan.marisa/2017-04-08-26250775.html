BoB 是这两年我相当喜欢的一个 Conf, 主要因为内容好多是 Haskell 相关, 并且比较偏实际应用, 还有几个我比较喜欢的 Haskell社区的人会开讲, 例如 2016 有:

玩 Haskell Type-Level Programming 跟 Generic 绝对很熟悉的 Andres Löh - Type the web with Servant!

Servant 主要作者, 搞 Type-Level Programming 也很多好玩 idea 的 Julian Arni - Verdict - Reified refinement (我相当喜欢的一个库)

Reactive-Banana 作者 Heinrich Apfelmus - Functional Reactive Programming

今年的 BoB 的视频在最近也完全放出来了:

John Hughes: Why Functional Programming Matters 在各大 Conf 都讲了一遍的 talk, FP 论文中经典中的经典, 再推荐一个也不为过 :D

Franz Thoma: Shell-Scripting mit Haskell 这是Gabriel Gonzalez的 Turtle 的介绍, 加上 Stack 出来之后 Haskell 做 Scripting 的方便程度几乎可以比得上 Ruby/Python/Perl

Immutable Hash Maps: Plenty of room at the bottom Functional Map (dictionary) 的实现 HAMT(Hash Array Mapped Trie) 的加强版 Compressed Hash Array Mapped Prefix-tree (CHAMP)的介绍...感兴趣可以看 OOSPLA 2015 的 Paper : Optimizing Hash-Array Mapped Tries for Fast and Lean Immutable JVM Collections

Andres Löh: Write one program, get two (or three, or many) bidirectional programming + generic programming!
