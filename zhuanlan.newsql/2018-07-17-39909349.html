原文

7 月 28 日，第二届 TiDB TechDay 即将落地深圳。我们准备了干货 Talk、美食和你们心心念念的 T 恤、贴纸……想和你一起聊聊技术，吃吃喝喝，谈谈情面面基。

自由软件之父 Richard Matthew Stallman 曾说过：“如果你想为这世界做些什么，仅有理想是不够的，你需要找条通往目标的道路并走完。”

TiDB 选择的开源之路，让我们在正确的道路上可以快速奔跑，同时收获了一个充满能量又有趣的社区。目前 TiDB/TiKV 已经汇聚了 260 位来自全球各地的 Contributors，并在 GitHub 上获得了 17000+ Stars，这是我们遥远征途上的一个小小的成就，感谢社区帮助我们一起走得更远。

去年七月，我们在上海举办了首场 TiDB TechDay，用一整天的时间为社区小伙伴抽丝剥茧的讲解 TiDB 技术细节。今年我们除了 TiDB 技术分享之外，还带来了精彩的用户实践和社区小伙伴的贡献心得。希望这一天，能满足你对 TiDB 更深层细节的所有好奇。

欢迎大家加入这一场社区狂欢，和我们一起感受开源的魅力！
