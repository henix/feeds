KY技术部，是KY的技术支持部门。我们通过各种技术手段，提升大家使用KY产品的体验，并为各种创意提供实现支持。我们希望能够以各种技术的优化，能让大家更精准的匹配到适合自己的朋友，能够找到合适的咨询师，能够与KY的交互中找到真正的自己，玩得意犹未尽。


本次我们招聘需求详情如下：


· 测试工程师 ·


工作内容：

1. 负责公司APP、小程序、H5等的测试；

2. 根据产品需求制定测试方案，设计测试数据和编写测试用例；

3. 收集整理各种产品问题，及时反馈并跟踪问题的解决；


任职要求：

1.  本科及以上学历，计算机相关专业背景；

2.  3年以上移动互联网端测试经验，熟悉移动互联网工作流程，熟悉bug管理系统及常用测试技巧；

3. 有独立分析分析问题和解决问题的能力；

4. 具备良好的沟通能力和团队协作精神，工作积极主动，责任心强。


· 产品经理 ·


工作内容：

1. 根据公司的发展策略，制定APP产品发展规划，负责APP产品策划、架构、流程等并完成相关文档；

2. 与研发、运营等团队进行沟通确保对产品的理解，负责推进开发进行开发工作，跟进测试和上线；

3. 负责产品及体验的改进和优化，提升用户体验；

4. 跟踪了解产品市场需求及发展趋势，充分了解用户需求，做好竞品分析；

5. 通过数据不断改进产品功能和体验，提升用户活跃度和留存率。


任职要求:

1. 本科及以上学历，5年以上互联网产品工作经验，有互联网教育经验者优先；

2. 有良好的逻辑思维与分析能力，能够规划和设计产品功能、使用流程与交互体验；

3. 具有良好的交流沟通能力，敏感的数据意识、善于发掘和分析客户需求；

4. 对于用户体验及UI设计有很好的把握；

5. 熟练使用产品设计软件Axure、PPT、Visio等等；



· PHP高级工程师 ·


工作内容：

1. 负责产品的需求分析、后端详细设计，构架设计以及开发工作。

2. 负责APP/H5/小程序 API系统和运营系统的开发和维护，并对现有产品进行优化；

3. 负责重大技术难题攻关，技术探索和分享；

4. 负责系统文档编写，并对通用技术进行整理，提高技术复用；

5. 少量的运维工作；

 

任职要求：

1. 5年以上相关工作经验，有大型互联网开发经验优先；

2. 熟练掌握PHP开发，至少掌握Yii/Yaf/TP框架中的其中一个，了解Swoole扩展及相关多线程开发经验者优先考虑；

3. 熟练掌握数据库相关开发，熟悉MySQL、Redis等不同类型数据库的开发；

4. 熟悉Linux服务器，熟悉常用shell命令，熟练掌握调试及问题分析技巧；

5. 擅长分析问题，有较强的创新意识与责任感，具备从前到后分析业务异常问题的能力；

6. 具备良好的沟通能力，对互联网常见业务模型熟悉。



· 前端高级工程师 ·


工作内容：

1. 与产品经理、设计师和后端工程师紧密沟通，理解产品设计意图并准确实现需求；

2. 参与公司小程序、H5、PC网站开发；


任职要求:

1. 5年以上互联网/移动互联网前端工作经验，熟练掌握HTML、CSS和JavaScript等前端技术； 

2. 熟练掌握jQuery、bootstrap等框架，熟悉HTML5特性，了解HTML5最新规范，能够熟练运用HTML5特性构建移动端的Web APP；；

3. 对各种浏览器的兼容性有一定了解，并有相关问题的解决经验；

4. 了解Node.js、AngularJS、React优先考虑；

5. 有良好的编码习惯，代码风格清晰, 简单, 保证运行效率；

6. 对用户体验、交互操作流程、及用户需求有深入理解者优先；

7. 简历请附作品（网站及相关作品）



· 公司福利 ·


· 五险一金、弹性办公时间、自选零食、年终奖

· 带薪年假、姨妈假、年度体检

· 明亮舒适的办公环境，交通便捷，吃饭方便

· 扁平化管理、工作氛围轻松

· 定期员工培训和团建

· 高速成长的机会



* 工作地点：上海市长宁区
