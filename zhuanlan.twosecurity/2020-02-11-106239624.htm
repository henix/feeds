InjuredAndroid

这是一个易受攻击的 Android 应用，与 CTF 靶场类似，带有 flag 及一些提示，可用于学习和认知 Android 应用漏洞。

通过它，你将认识到一些 Android 应用中的漏洞。

项目地址：https://github.com/B3nac/InjuredAndroid

包含九项练习：

  • XSS 练习
  • 登录01
  • 出口
  • 资源
  • 登录02
  • 广播
  • 登录03
  • SQL 练习
  • AWS

以及 flag 统计。

安装

两种方法，一种是在物理机上进行安装，另外一种是在 Android 模拟器上进行安装。先讲讲怎么在模拟器上进行吧。

模拟器上进行安装

首先，下载 apk 文件。

下载地址：https://github.com/B3nac/InjuredAndroid

然后从 Android Studio 中启动模拟器（使用带 Google API 的模拟器，以便开启 root adb）。

最后将 apk 文件拖入模拟器安装即可。

物理机上进行安装

这个比较方便，同样先下载 apk 文件，再用数据线将手机与电脑连接，手机开启 USB 调试模式，然后使用 adb 将 apk 安装到手机上。

adb install injuredandroid.apk
或者直接发送到手机进行安装。。。

一共 9 项挑战练习，都是针对 Android 应用的漏洞练习。
