原文

上篇文章「为你的电子书制作精美的PDF」得到了不少读者的反馈，很多人建议我看看gitbook，同样是一个生成电子书的利器。对于gitbook，很久前我在hacker news上看过介绍，后来没怎么使用便渐渐淡忘。这次重新拾起，发现自己的osx里竟然已经装好了calibre（gitbook生成pdf所需的工具），简单试用了一下，感觉非常不错，生成的PDF效果也很不错。我的下一本书，会考虑使用gitbook的。gitbook有两个问题：

1) 生成PDF的速度略慢，我自己的测试：

➜ git git:(master) ✗ time gitbook build -f ebook Starting build ... Successfully built! gitbook build -f ebook 54.48s user 0.63s system 90% cpu 1:00.96 total

2) 我没有找到自定义样式的方法，修改缺省的PDF样式要比princexml的方案麻烦。如果有读者知道如何简单/快捷（而非fork gitbook，然后修改源码）地定制生成的PDF样式，烦请告知，程序君在此先谢过了。

言归正传。

当你完成一件事情的时候，总结是个不可忽视的活儿，这也非常符合build-measure-learn的思想。我的「途客圈创业记」可以算是我创业的总结 —— 虽然这总结的时间跨度略长，有一年半之久；而这篇文章，也可算得上我写完「途客圈创业记」的诸多总结中的一个。

这个总结想探讨的问题是：如何把一本书产品化？

把个人作品当成产品来思考是我在读了 “lean startup” 一书后做出的一个决定。此后，每个私人想法，我都会使用lean startup canvas分析一下，然后再执行。对于这本书而言，我们先从lean canvas里面的几个要素，看看这个产品的特点。篇幅关系，我就只讲讲顾客区隔，独特价值主张和营销渠道。

顾客区隔（Customer segment）

写作之初，我把这本书的目标用户就定义为对互联网创业感兴趣的读者，打算创业，或者正在创业的读者，包括但不限于：

1) 大学生

2) 各科技公司的技术人员，产品人员，运营人员

3) 对目前平淡的工作和生活感到乏味，想了解不一样的世界的人

这个目标人群基本上抓准了。学生和科技公司的职员自不必说，对于第三类目标人群，我也找到了很多很多有共鸣的读者。在每次连载中，都有一个个陌生的身影跳出来，跟我说她/他虽然这辈子可能都和互联网创业没有交集，但很喜欢这样的故事。

独特价值主张（Unique value proposition）

要论创业的经验之谈，本书离孙陶然的「创业36条军规」十万八千里；要论思想和内容的多元化，知乎出版的「创业时，我们在知乎聊什么？」甩开本书好几条街；如果说思想的深度和高度，那「四步创业法」，「精益创业」等名家之作更是我这样一个小辈所无法企及的。

但是，本书的价值在于作者的亲历与反思。本书完整如实地记录了我在途客圈创业的历程，有如一次复盘，除了讲述整个创业历程中的种种经历外，还提供了很多个人的思考在里面。

这个独特价值是否被认可，有待于市场进一步的反应。

营销渠道

我选择「百度阅读」作为本书的销售渠道。自6月25日连载，截至到目前，有超过25000免费读者，却只有不到1%的付费读者转化率。我尝试使用个人的微信公众号，知乎专栏作为免费的载体来宣传付费图书，然后通过调节免费/付费的增值服务，来吸引更多的用户付费阅读；于此同时，百度阅读通过首页推荐，周末特价，公众号宣传等一系列活动来激发本书的销售。

其实豆瓣/多看等电子书渠道也可以成为备选的营销渠道。选择百度阅读有我的道理（对于一个首选平台的选择，也许我会考虑单独撰文阐述），没进一步将渠道扩展到豆瓣/多看主要是个人精力的问题。

通过免费拉动付费是个正确的策略，尤其对我这样的毫无名气的新手作者。在扩大免费读者群的同时，提高付费阅读的转化率就是一个很重要的营销指标，而这其中，二者的差异化，或者说，付费带来的增值服务（享受）就成为一个关键。

一开始我对电子书销售的认知是：价格对大家购买与否有直接的关系。所以，「途客圈创业记」订了一个3.99的低价，也就是说，一根雪糕，就能换来我这辛辛苦苦码出来的15万文字。但是随着各种营销手段的实验，我发现，价格并不是大家购买一本电子书的首要推动力 —— 即便将这个价格7折，也并未激发更多的免费用户转化为收费用户。

在连载完成之后，我做了一次价格的调整，3.99变成了6.99，一根雪糕变成了一杯圣代，或者，一袋75g的乐事薯条。销量并未随着价格的增加而降低，反而，有那么一两天，日销量都过了5本。也是，这钱，在肯德基或者超市的货架前，也就是一挥手的事。

我打算接下来在全部文字修订完毕之后，在百度阅读里更新并捆绑本书的PDF，然后把价格调整到9.99，继续测试在扩大免费/付费之间的差异化后市场的反应。当然，这PDF并非普通的PDF，而是为百度阅读的读者私人订制，至于怎么个私人订制，且容程序君卖个关子先。^_^

（当然，如果你还有什么更好地将自己从免费读者转化为付费读者的方法，或者，你有某个和本书相关的心愿，了此心愿能让你愉快地购买，也请告知哈。）

这样的阶梯定价我会不断尝试，不断完善，在自己的下一本书也会继续使用。背后的思想是：

当你用3.99购买我的电子书时，你买的是一份风险和等待。一本处在连载中的书需要你足够的耐心和接受可能的断更的风险。

当你用6.99购买时，你买的是一本完成的书稿。你可以一口气从头读到尾，怎么爽怎么读。

当你用9.99购买时，你买的是一本完成的书稿和一份增值的服务。这增值可能是一个内藏惊喜的PDF版本的电子书，也可能是旁的什么。

因为承受了一定的风险，所以3.99/6.99的用户都能享受9.99的用户能享受到的一切。

先谈这么多。电子书一直被认为是纸书的终结者，我对此深信不疑，只不过，作为一个乐观的悲观主义者，我觉得，电子书终结纸书这事在国内还有很长很长一段路要走。但是，电子书是可以玩出很多花活的，且行且实验～

别怪程序君没提前告知哈，趁着现在还6.99元，赶紧订购「途客圈创业记：不疯魔，不成活」吧。等到私人定制版本的PDF完成并放置下载链接中后，这本书的价格会提至9.99哦，亲！
