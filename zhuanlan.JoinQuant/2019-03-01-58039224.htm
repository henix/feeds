北京 / 西安

清华大学交叉信息核心技术研究院

交叉信息核心技术研究院是由世界著名计算机学家、图灵奖唯一华人得主、美国国家科学院院士、中国科学院院士、清华大学交叉信息研究院院长姚期智院士亲自负责，清华大学交叉信息研究院近30位教授参与研究并运营。

公司三个研究方向分别为金融科技与监管科技研究中心、前沿架构与智能芯片研究中心、城市大脑研究中心，和西安市在政、金、产、学、研五个领域开展合作。

智能金融核心引擎系统项目综述：

本项目依托清华大学交叉信息学院，交叉信息核心技术研究院，金融科技与监管科技中心，为核心金融机构打造智能金融的决策核心引擎，在智能资产管理，财富管理与智能交易方面输出世界一流的前沿技术，正在与国内外多家头部券商、银行、保险资管公司进行战略性合作。本项目利用人工智能方法，统计学习，计量与金融工程方法对经济金融市场进行大规模建模与场景模拟，开发高效抗噪音的大规模金融优化算法，为投资者、金融机构提供智能化决策引擎, 也为监管机构提供决策辅助系统。

工作内容：核心引擎的算法升级研发，系统开发与产业实践

机构提供：以下工作机会皆招聘全职成员与实习成员。全职成员提供有竞争力的薪资和相应的产业化机构期权激励，实习成员提供实习薪资与转正机会。

软件架构师（实习/全职）

职位描述：

  1. 参与技术架构设计优化，根据业务规划及技术规划制定应用架构方案；
  2. 核心架构代码实现和技术难题攻关；
  3. 指导并提升团队整体开发能力。

职位要求：

  1. 计算机或软件工程相关专业本科及以上学历，丰富的项目开发经验；
  2. 深刻理解计算机原理，有良好的数据结构和算法基础，能够编写高质量代码；
  3. 熟练掌握python、Java等编程语言，熟悉常见设计模式；
  4. 熟悉微服务、SOA架构、容器、持续交付、自动部署；
  5. 熟练掌握MySql/Redis等相关数据库技术；
  6. 熟悉Linux/Unix开发环境及分布式系统；
  7. 有以下基础支持或技术开发经验优先：大数据相关基础、机器学习基础、开源项目开发贡献等。

算法工程师（实习/全职）

职位描述：

  1. 根据产品的需求，完成算法设计、实现和调优，推进算法落地；
  2. 跟踪前沿技术的发展趋势，并开展算法在金融量化、投资等领域应用研究。

职位要求：

  1. 数学，统计，计算机科学等相关专业本科及以上学历，硕士博士优先；
  2. 熟练掌握常用的数据挖掘、机器学习、深度学习、推荐算法，至少精通某一方面；
  3. 具备强悍的编码能力和扎实的数据结构和算法功底，掌握python语言；
  4. 有大规模海量数据机器学习/数据挖掘相关经验者优先；
  5. 了解基础金融知识、有相关实习经历优先。

优化算法工程师（实习/全职）

职位描述：

主要进行资产组合优化领域的研究。包括: 二阶锥优化、资产组合稳健优化，考虑交易费用的非凸优化，多目标多优先级优化，相关功能代码的实现。

职位要求：

  1. 运筹、统计、机器学习等相关领域硕士以上学历， 博士学历者优先；
  2. 对优化算法领域有研究者优先；
  3. 能对英文文献进行快速阅读、总结、ppt 展示， 有较强的沟通能力和责任感；
  4. 熟练掌握python/R /matlab 至少一种编程语言。

量化建模研究员（实习/全职）

职位描述：

主要进行金融市场建模相关方向的量化研究。包括:对股票，利率与债券的因子模型建模，期限结构匹配与资产负债管理相关研究，风控管理与业绩归因，相关功能代码的实现。

职位要求：

  1. 统计计量，金融工程，金融数学，数学，物理，计算机等理工科硕士（含）以上学历；
  2. 能对英文文献进行快速阅读、总结、ppt 展示， 有较强的沟通能力和责任感；
  3. 熟练掌握python/R/matlab至少一种编程语言；
  4. 对金融计量，随机分析， 概率统计方面有研究者优先；
  5. 有金融机构研究岗工作，实习经验，或通过 CFA/FRM 考试，有一定金融背景知识者优先，但不做为硬性要求。
