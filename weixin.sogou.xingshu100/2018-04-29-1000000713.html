发现最有创造力的思想。


艺术的终极目标是触摸生命，穿越生死的甬道，让我们久别重逢。所有伟大的艺术，都在给我们这个机会。


艺术源于生活的升华，在升华后又具有纯净的灵性。如今的生活节奏非常快，但是有艺术的相伴，生活应该慢下来，去享受其中的乐趣。


艺术不是个体，它是人与人，人与生命的连接，它让人不孤独，让人感觉到人，让情感累积，引起共鸣。


什么才是触达我们生活的核心内涵？——人文价值。那是人类所有最先进、最灿烂、最优秀的文化诞生的根本。


今天，4月29日，上海，造就联手SMG LIVE，开启人文系列的首部年度大片——《造就之舞，感知与经历的漩涡》。


请看本场精彩回顾。


{  陈少琪  }

我给哥哥作词的收官作是《风再起时》




“任旧日路上风声取笑我，任旧日万念俱灰也经过，我最爱的歌最后总算唱过”……那些贯穿香港三代人记忆的流行歌，原来都是他写的。这一次，陈少琪和我们分享他的乐坛经历及创作之路。


1986-1995年开启创作之路，当时给达明一派作词，受到前辈歌手的关注。


2005-2007年进军内地，彼时内地音乐开始腾飞，无奈水如不服颇受挫折，当时人已40，重新起步太老，而退休又太早，庆幸2008年为电影《画皮》创作主题曲《画心》，一举进入内地观众视野。


而后幸运地从作词人走向音乐制作人，为迪士尼创作主题歌曲。


最情有独钟的表演方式是音乐剧，因为这是一种跨界的艺术。我有一种天分，很小的时候就能把文字转换成歌曲，到了初中又可以把英文歌曲转换成中文歌曲，直到后面读平面设计，又把这种能力转换成音乐、摄影、制作等等。


我很喜欢把生活中诸多不一样的体验，创作成作品。艺术是相通的：受朋友启发，参与摄影展览；热爱素描，从黑白中收获纯净心情；和自己心灵对话，从中构想自己的作品。

艺术是生活的升华。如果你看到平凡而无特点的东西，你是否能从中看出不一样的地方？我很喜欢从平凡中看到不平凡。


勤奋是创作的关键，香港乐坛的黄金一代均具有这样的品质，勤奋是你能在这个圈子里生存下来必须的条件。


对于乐坛新人来说，勤奋是崛起的希望，互联网时代的艺人不仅需要人设，也需要艺术修养。



给乐坛新人的寄语：勤奋是崛起的希望、艺术常青的关键。


{ 高山峰 }

梦想的力量，无微不做



你还喜欢听广播吗？你还记得那些年我们听过的电台情歌吗？《东方风云榜》原来是由他主持并担纲晚会总导演。这一次，高山峰和我们分享电台从业的宝贵体验。


12岁的爱好，用激光笔、手电筒、奶瓶、灯泡等等搭了一个“舞台”；今天实现了自己的梦想，成为了真正的舞台导演，导演华语界重要的音乐颁奖礼——东方风云榜。


当年参与东方风云榜的工作，还是一个打杂的“小朋友”。那些年的最受欢迎歌手榜单，都是人肉“跑”出来的“数”据，是当时的“小朋友们”用人力一票一票数出来的。


从最初的数票数，到接待艺人，到全程打造整场活动；从最小最细微的工作开始，到创建整个舞台……这是一段奇妙的过程。


坚持不懈的电台追星经历：给电台偶像写信，打电话，最终打动主持人，1997年进入广播大楼开始参与节目。初入职场，从最基础的工作做起，从“小朋友”做起，从拿外卖、拆信做起。


2006年，终于有了自己的第一档节目，收获了“听节目长大”的观众。通过微笑，传递积极的表达。


2011年电台的变化：从主持人自主放音乐到电台编排所有音乐，由听众决定曲库。广播人的坚持：把每一首歌放完整，回复每一条收到的私信。


广播最大的魅力在于，你永远不知道接下来会播放一首什么样的歌曲。


{  陶冶  }

舞者的每一寸每一瞬，都是生命力的绽放




你真的了解你的身体吗？从生到死，身体才是我们理解万事万物的载体。在重力的枷锁下，我们的身体如何突破极限，抵达生命中最单纯的圆满？这一次，陶冶为我们讲述那份舞台上流逝的永恒。

 

舞蹈作为原始的艺术，有人称之为艺术之母，我认为它完全可以称得上，因为舞蹈是以身体表达，舞者的每一寸每一瞬，都是生命力的绽放。


 

舞蹈是没有一个捷径的艺术行业，通过朴素的身体力行来做呈现。身体其实是一个泛艺术的概念。身体可以是剧本、是建筑，更是承载生命的本源。舞蹈是身体的表达，时间的表达，语汇的表达，也是生命力的表达，它连接现场，连接舞者和观众。

 

我不仅分享舞蹈的美，身体的力量，更关注运动的真实。运动研究是陶身体剧场一直摸索的方向。身体是有限的、清晰可见的肉身，但通过舞蹈，可以展示千姿百态。同样的动作、次序，会有不同的展现。


身体运动的时候，每一个过程都在做抛力运动，这是关于生命的对话。舞蹈，作为探索身体本身力学的入口，旋转、失衡，身体就像在一个宇宙的空间中，没有支撑，这对舞者身体的开发有极致苛刻的要求。

 

一个动作出现后，像抛出问题，下一刻要解答这个问题。舞蹈就是来回提问和解答问题，就像思辨的过程。

 

舞团里没有男性、女性的差别，中性是最强大的。雌雄同体，你可以非常有力量，也可以非常柔软。


 

瞬间即永恒，舞蹈就是这么一个遗憾的艺术，每一个瞬间都会逝去，我一直告诫舞者不要迷恋在舞台上那一刻的表达，当大幕落下，你又变成一个孤独的个体。

 

要感受每一刻，要感觉自己与这个世界的联系，当在你舞台上，所有的一切共在。无数个瞬间累积在一起，形成流逝的永恒。在这个过程中，身体成为时间的物化，身体就是时间的度量。


 

孤独是人终极的恐惧，就如同害怕死亡。但舞蹈让人的情感凝聚在一起，引起共鸣。


{  张军  }

你迟早会爱上昆曲，它是我们触摸生命的一次机会




台上人浅唱低吟的正是台下一对对眷侣并肩缱绻的那一刻，从古至今，人们内心深处对幽微曲折情感的诉求都未改变。若你还没未爱上昆曲，一定是还不够了解它。这一次，张军为我们解读昆曲之美。


昆曲是百戏之祖，被誉为“中国戏曲之母”，它有着600多年的历史，从江苏昆山发源。余秋雨称，昆曲主载中国审美长达200多年。昆曲被称为中国的莫扎特、贝多芬和巴赫。


昆曲受到严格独特的要求，每一句都蕴含典故。你迟早会爱上它，只是你还没找到那把钥匙。


我进入昆曲学校三年后还不知道为什么要唱昆曲，直到我学会了这一段：月明云淡露华浓，欹枕愁听四壁蛩，伤秋宋玉赋西风，落叶惊残梦，闲步芳尘数落红。《懒画眉》让我学会了柔绵、静谧、浪漫……从而开启昆曲之路。


20年来我们只做了一件事，深入你们当中，分享昆曲是什么。即将于518举办的「水磨新调」新昆曲万人演唱会，是要用今日音乐人情怀去唤醒过去的记忆。


与其说我们发了宏愿要站上万人舞台，不如说我们在过去20年你们的支持下充满了勇气，期待能在未来20年继续守护我们共同创造的景象。让我们一起行动起来，为昆曲留下些什么吧。
