认识你自己系列终章：低自尊



过年前几天，社群里的一个朋友问了我一个问题，故事的背景是这样的：


这位男生A和女生B打电话，女生B说到心中的痛处哭了起来，本来应该是女孩真情流露，男孩细心呵护的情节，可惜事与愿违，男生A的手机突然没电自动关机了。


为此，男孩很担忧，问我该怎么和女孩解释。


这个问题相当的诡异，因为答案太简单了，只需要简单的陈述手机没电的事实就可以。


既然如此简单，为什么这个男生需要来询问我呢？那肯定还有更深层的需求。


我回答男孩：你实话实说就可以。


男孩给我截了图，说，已经实话实说了，现在想知道怎么补偿她。


“补偿她”？


这个问题又很诡异，这并非是故意伤害，为什么需要“补偿”？而且是否造成伤害，现在还不清楚，即使真的需要补偿，不是说一句对不起就可以了吗？


这个男孩提问的时候，我能量比较充足，并没有简单的觉得无聊，而是对这背后的原理产生了兴趣。


我试着将自己代入某一种特别敏感焦虑的状态，比如，一个我特别喜欢的特别优秀的女孩子，本来对我不大待见，但是却突然和我打电话了，当她说到动情处，我的电话没电了，这时候，我在想什么？


这样就很清晰了。


我的电话突然没电→她会因此不开心→她会对我失望+我的手机居然会突然没电=我这人不靠谱→我希望补偿她=我希望改变她对我是不靠谱的看法=？


所以，这个男生真正要补偿的是？


A. 女孩对男孩的看法

B. 女孩的情绪

C. 男孩认为女孩对自己的看法


你觉得答案应该是哪个选项？


是C。


男孩要补偿的，不是女孩，不是女孩对自己的看法，而是，我自己认为的女孩对我的看法。


所以，归根结底是讨好自己，


深层次情绪：自卑。


因为自卑，所以对别人对自己的评价持悲观预期，所以觉得女孩会对自己失望，所以需要做点什么补偿。


想到这里我有点头疼，要怎么回复他呢？要让他现在的情绪消退需要让他接纳自己此时此刻的自卑，这很难，正当我想着怎么整理语句的时候，他回复我了：


不好意思，极乐老师，事情解决了，刚才是我自卑发作了。


看到这个，我非常欣慰，能够主动意识到自卑，自卑就不再会困扰你，我觉得，如果有一天我死了，我的碑墓上一定该有这句话：


他的一生，影响了不少人。


这是认识你自己系列的最后一章，之所以写这个系列，是因为我这么多年的工作经验让我认识到：


给人们带来痛苦的，不是“你的问题”，不是啥原生家庭父母不好，不是啥缺爱，不是啥性格诡异，而是你对自己的问题——一无所知。


终章：低自尊


低自尊是一种状态。


可能很多人都有这样的体验，我和朋友玩耍的时候，我觉得自己很放得开，很正常，但是一旦和父母交流，就总是要争吵。


原因就是，当我们面对父母的时候，我们处于一种低自尊状态。


女孩子这点的体验应该更深刻，恋爱初期，他追我追得死去活来的，我觉得自己棒棒的，从来没什么情绪，知道照顾他，但是随着时间推移，看着他越来越优秀，而我自己在原地，就会想东想西，觉得他不爱我了，觉得他外面有人了，明明知道这些想法都是自己自找的，但是就是克制不住。


这也是一种低自尊状态。


低自尊往往伴随着讨好、攻击、自责、拧巴... ...情绪，我们往往能看到自己的讨好、自卑、攻击、愤怒，但是我们并不具备感受自己处于低自尊的能力。


那如何改变这种状况呢？


大学的时候，我的老师强制要求过我们阅读《自尊的六大支柱》，并且做读书笔记：3w字。


那时候看这书啊，觉得真几把无聊，有这穷功夫还不如玩两把war 3呢，最初我们看这书的时候，还没有出版，是课代表和班长帮大家打印的版本。


这些年下来，越来越发现，所有人都需要读一读这本书，前几天我再次翻看这书，依旧还是和大学时候一样，读着拗口和无聊，心理学家的思维方式已经和正常人不一样了，所以心理学家的书，大多都被束之高阁了，可惜了。


我找到这本书的总结版本，可以让大家先看一下目录，看起来朴实无华，没什么特别的，但是内涵都是异常丰富的，我会做一个系列，用5-6篇文章，诠释自尊的自我养成方式。


目录：


支柱一. 有意识的生活


1. 我越清楚与我利益、价值、需要和目标相关的事物，我的生活就越成功。

2. 经常使用大脑其乐无穷。

3. 改正错误比掩盖错误于我更有益

4. 注意区分事实、解释和情感

5. 我应时刻警惕，抗拒自己规避现实的念头。

6. 如果我的眼界更宽，我的生活和行动将更具效力。


支柱二. 自我接受


1. 我是为自己而活。

2. 我接受自我，并接受自己思想存在的现实。

3. 我不一定要喜欢、赞同或受自己情感的控制，但我能接受它们的存在。

4. 即使我后悔、自责，但做过的事我会承认。

5. 我承认我所思、所感、所为均是自我表现，至少当它们发生时如此。

6. 我承认自己有很多问题，但我不受它们的束缚。我的问题不是我的本质，我的恐惧、痛苦、困惑、或错误不是我的核心。


支柱三. 自我负责


1. 我为自己的存在负责。

2. 我为实现自己的欲望、幸福负责。

3. 我为自己的选择、价值观和行动负责。

4. 我为自己以何种意识层面进行工作和参加各种活动、与他人交往负责。

5. 我为自己如何安排时间负责。

6. 我为提高自己的自尊感而负责，别人无法给我自尊感。

7. 从终极意义上说，我接受自己的孤独状态。在一些具体问题上，人们可以给我一些帮助。但没有人能担负起我生存的最基本的责任。

8. 自我负责的需要是很自然的。我不认为它是一场悲剧。


支柱四. 自我保护


1. 我可以尽情地表达自己的思想、信念和情感。除非在某种场合中我觉得不说为好。

2. 我有权维护自己的信念、价值与情感。

3. 让别人认识、了解我于我有利。


支柱五. 有目的的生活。


1. 只有我自己才能选择生活的目标，没有任何其他人能设计我的存在。

2. 我若想成功，就必须学会如何去实现自己的目标。我需要自己规划并完成行动计划。并关注自己行动的结果。

3. 为了自我利益，我应高度审视现实，即注意承载我的信念、行动和目的的信息和反馈。

4. 对于自律，我不应视之为“牺牲”，而是实现自我欲望的一种自然的先决条件。


支柱六. 个人诚实


1. 我应言行一致。

2. 我应信守诺言。

3. 我应公平、正直、善爱、热情地对待他人。

4. 我应努力保持道德前后一致。


总结：支持自尊感对现实的信念


1. 是即是是，事实就是事实。

2. 自我蒙闭双眼不能使真变假，也不能使假变真。

3. 尊重现实（尽自己最大努力去把握它们）比漠视现实更能带来硕果。

4. 生存依赖于正确地运用意识、忽视意识的责任不能提高自己的适应力。

5. 从原则上说，意识是可靠的，知识是可得的，现实是可知的。
