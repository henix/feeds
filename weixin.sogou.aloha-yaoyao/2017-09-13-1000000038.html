遥遥的原创、译作和沉思录

新媒体管家

最近权游第七季完结

维斯特洛上的家族几乎团灭一半

但是很多简洁而富有力量的族语已经深得我心，例如


在欧洲的中世纪，凡是有点身份的家族

都要搞搞家徽，写写族语

那么这些东西，程序员也得有是吧

于是我就想了一些口号，把自己代入中世纪的感觉


CODE FOR NEXT GENERATION

码为下代，码通未来

这里的 code 既是动词也是名词

作动词时，是为下一代写代码（母婴APP）

作名词时，指的是通往未来的关键（key）


CODE DANCES, LIFE FLOURISHES

码动不止，生命不息

编程被视为“指尖的舞蹈”，所以用了跳舞一词来形容

而程序让生活变得更具有生机

让生活的选项如花园般丰富多彩、生生不息


WE CODING, WE LIVING

我码故我在

来自哲学家的思考 I think, therefore I am 一句

程序员普遍有被认可的心理欲望

大多数情况下是通过编程作品得获得他人肯定

故有感而发


COMMIT CODE, COMMIT FUTURE

码许未来

commit 是提交写好的代码的意思、也是承诺的意思

所以提交了完成作品

也就相当于承诺了会发生的未来，一语双关


BABY GROWS, CODE LIVES

宝宝长成，代码永生

来源于“铁打营盘、流水新兵”的想法

代码也许不会变，但是会陪宝宝一起长大

在这个过程中为宝宝带来更好的生活

从而代码的价值获得永生


FUTURE EMERGING AS CODE ROARING

万码奔腾，未来已至 

向未来，共生长

Nothing but the Future


CODE HARDER, MOMMY HAPPIER

让妈妈开心的代码，码了又码

来源于苹果式中文

开发者的大事，大快人心的大好事

哈哈哈哈哈哈


MONKEY RISES

猩球崛起

告诉大家一个秘密

猩猩和程序猿其实是同一个物种

当然和攻城狮也是同一个物种
