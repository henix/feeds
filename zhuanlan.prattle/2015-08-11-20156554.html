原文

使用Soylent作为我的唯一食物来源有一周多了，本来想测试两周，把订购的14袋soylent消耗干净再写这个测试报告，无奈朋友圈的人三天两头询问我感觉怎么样，是否还活着 -_- 不胜其烦，只好提早发出这份报告，供大家参考。


既然是个测试报告，那么test case必须要有，就功能上，我设计了如下测试例（直接附测试结果）：


  • 一天只吃soylent，分三次吃，早7:30，午12:00，晚6:00。看身体的反应：中午11:00左右开始感到饿，下午5:00感到饿。其他并无不适。

  • 一天只吃soylent，分四次吃，早7:30，午11:30，下午3:30，晚上7:30，隔4小时一次。看身体的反应：这个节奏很适合我，这段时间我基本都是这么吃的。

  • 一天按三顿的量吃两顿soylent，余下的一顿以四根香蕉补充。看身体的反应：四根香蕉在晚上6:00吃下，晚上11点饿醒了，吃了两片面包才睡去。

  • 隔48小时服用之前冲好的soylent，看看有没有问题（soylent手册建议冲好后两天内使用）：口感上没有任何问题，吃着也没有任何不适。

  • 冲好后立即食用，冲好后放微波炉里加热（微热）食用，冲好后放冰箱三小时后使用，看看口感：在常温或者微热的情况下，口感很差，粗糙，喝着不舒服；长时间冷藏后口感很好。

  • 冲好后尝试一顿喝1/2以上，看看身体的反应：中午喝的，喝到接近1/2的时候，饱腹感很明显，喝到1/2已经不想喝下去了。之后打了好一会的嗝。

  • 一天喝两袋，看看身体的反应：没测。做了上面的测试后，完全没有心情做这个，身体直接抗拒。

  • 喝完七袋以后，检测一周以来的体重变化：一周前第一次喝之前裸重75.6kg，今日74.3kg。期间跑了53km（29 + 6 + 18)。所以体重的下降属于正常。

  • soylent + 香蕉 / 混合莓（蓝莓，草莓，树莓） / 苹果 的口感：加了香蕉的口感不错，加了混合莓（costco买的袋装品，非新鲜水果）的颜色怪怪的，味道也偏酸，不过也可以；加了苹果的口感一般，至少我不喜欢。混合香蕉的最大问题是静置久了颜色会变，最好尽快喝掉（打的时候加冰块）；而且放太久，味道也不好，有种熟过劲的香蕉的感觉（其他新鲜水果也有类似问题，不过香蕉最明显）。

由于这是功能测试，不是压力测试，所以一切不把自己的身体当身体的测试都没有进行。


首先说身体的反应。一袋soylent提供的2000KCal热量，对于像我这样的不吸烟不酗酒不怎么吃零食体态匀称的正常成年男子来说足够。对于那些不吃蔬果便会感觉营养不够的同学，可以尝试在soylent里面添加水果，甚至蔬菜（我同事喜欢没事搁点芹菜，我没敢试），完全没问题，而且口感会更好一些（因人而异）。做法很简单，当你手边有料理机的时候，把水，蔬果，soylent按适当的比例加进去搅拌就可以了，做的时候最好加冰块，这样可以在蔬果新鲜的状态下立即食用。



起初几天我有过消化不良 —— 肚子里总感觉有气，总想上厕所去什么也没有。这可能跟我没有按照说明书的建议，逐步增加soylent的供应，而是一上来就全天食用导致的。现在我偶尔还有胃胀气，但感觉轻了很多。


其次是大家都关心的问题：喝完会不会很容易饿。这个完全不用担心，喝完后会有明显的饱腹感，不用担心流食不抗饿这种以讹传讹。soylent冲好后本身就比较粘稠，适合在胃里缓慢消化，而其配方里面又含有足够的蛋白质（能提供25%，也就是500KCal的热量），而蛋白质的消耗是一个缓慢的过程，所以人体不会有饥饿感。


再说说口感。不加任何料的soylent 1.5口感非常不错，类似于豆奶+燕麦的味道，不甜，入口丝滑，非常舒服。7天下来我并没有感到任何不想喝的时候（作者注：在二十多顿soylent里，只有几顿是加了水果的，其他都是原味，所以这个感觉是可信的）。就算原味吃烦了，加上果蔬，甚至老干妈（谁来试试哈），又是另一种感觉。


之前关于soylent我也提到过，它最大的好处就是方便。这种便利直到你亲身体会时才有感触。有一个周日我想全身心投入在12个小时内免打扰地做点什么 —— 这使用soylent在之前，是完全不可能的。三餐不能不吃，否则胃肠非暴力不合作式的蠕动会打乱大脑的任何思考。即便是叫外卖或者吃准备好的便当，总还是有吃的时间和收拾的时间，干扰性很强。soylent把这些繁琐简化成从冰箱里拿一个瓶子，倒足够分量在杯子里，然后在思考的间歇喝一口两口这样一个几乎不干扰大脑的进食方式。所以，单凭这样一个好处，就可以让我屯上一些soylent，以备不时之需。


然而，soylent也有一些局限。


最大的问题是它让你没了社交。这里的社交是指面对面的社交，而非微信微博这样虚拟的社交。基本上，我们社交的主要来源是吃饭：早餐和家人社交；午餐和同事社交；晚餐和家人或者朋友社交。soylent取代那些你独自一人默默进食的场景是非常合适的，但它无法完全取代除此之外的场景。食物是一种美妙的存在：它好看，好闻，能把人们聚到一起开怀畅谈。中午的时候，几个同事围着圆桌吃饭，聊花千骨何以笙箫默，你30s喝完午餐，然后尴尬地坐着那里，闻着饭香，饺子香，菜香，看着一颗颗在筷子尖上嘲弄般眨着眼睛的油花，等着大家一点点把便当盒里的东西享用干净，内心需要无比坚强。晚上或者周末，好久不见的朋友们商议去哪家馆子，而你咂摸着Soylent，残忍地拒绝，就这样不知不觉地将自己与大家隔离开来。如果连续完全靠soylent生存一个月，某种程度上，就是自己在这个社会上的一种自我放逐。


还有一个问题是它对消化系统的影响，当然，也是一个长期服用带来的问题。soylent的蛋白质来源主要是植物蛋白，以糙米和燕麦粉为主（注意，不是soy - 大豆哦，可能考虑到豆类过敏者不少），所以五行缺肉的朋友久了可能会很不适应。另外，长期靠流食生活，究竟会对消化系统，排泄系统带来什么样的影响，尚不清楚。


所以，经过一周基本上完全依赖soylent的测试后，我决定把自己的生活稍稍回归一下：除非周末我要把自己完全投入进一件事里，其他时间我一天顶多就来一顿soylent。这即保证了我能在某些我不想做的事情上犯懒，也还能让我自己不至于生活在一种「长期失联」的状态。我依旧喜欢soylent，喜欢它带来的便利，我会继续订购，继续推荐给我的朋友们；但我显然不会把它作为唯一的食物来源（事实上，soylent的发明者也不希望人们仅以soylent为食）。


soylent目前仅仅在美国和加拿大出售。由于欧盟对农业市场的严格保护，至今soylent无法触及那里。不过在欧洲的同学不必担心，哪儿有很多soylent的copycat，比如：mymana.eu，queal.eu，joylent.eu（看清楚了，不是joyent哦，我第一次也差点当成了nodejs的公司joyent），jakeshake.eu 等。至于国内嘛，我不是很清楚，有个 github repo stormzhang/1024，名字略恶心，叫 超模1024，看着不像恶搞，可以瞅一眼。
