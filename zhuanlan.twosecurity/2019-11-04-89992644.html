原文

网络犯罪正在不断的发展，并且不断变得更加具有破坏性。尽管攻击者可以利用的攻击媒介种类繁多，但勒索软件仍是网络犯罪最多使用的形式之一，它一直保持着作为今年主要网络威胁的头把椅的地位。

几乎每天都有类似的事件报道，大型公司或政府组织（尤其是在教育和医疗保健方面）被黑客入侵攻击，原因是基础设施薄弱，运行安全性差。当然，勒索病毒还造成了一些有史以来最具破坏性的攻击：像臭名昭著的 WannaCry 和 NotPetya 在2017年成为头条新闻。

WannaCry：一种“蠕虫式”的勒索病毒软件，大小3.3MB，由不法分子利用NSA（National Security Agency，美国国家安全局）泄露的危险漏洞“EternalBlue”（永恒之蓝）进行传播。
NotPetya：源自类似Petya的全新形式勒索病毒，也是利用永恒之蓝漏洞（EternalBlue)进行传播。

随着勒索软件攻击变得越来越复杂，对于各种规模的企业而言，采取主动的方法来保护网络安全变得越来越重要。当考虑到勒索软件可以采用的多种形式和输入方法时，这似乎是不可能完成的任务，但是企业可以通过审查其现有的安全策略并确保采用分层的方法来确保它们得到充分的保护。

与时俱进

网络安全必须始终保持一种鲜活的状态，旨在应对新型威胁时进行升级和评估。但是，随着安全措施的改进和变更，网络攻击者设计的勒索软件也将渗透其保护。只有保证不断得更新迭代，才能长久地存活下去。

勒索软件在今年正在进行战术复苏。Ryuk，Phobos，Grandcrab 和 Rapid 是最常见的企业攻击的顶级勒索软件系列。如你所见，勒索软件现在可以针对特定组织或所选目标进行定制化构建，以造成对其最大的损失，因此被攻击者需要支付更高的勒索值来换取安全释放数据。

攻击者继续通过易受攻击的计算机强制访问公司的内部网络，以获取有关业务中有价值的信息。然后，他们可以在了解公司大致情况后，拟出公司承受范围以内的赎金，从而使攻击收益最大化。即使是经验不足的黑客也可以创建破坏性极强的攻击，因为在暗网上可以购买定制化的勒索软件，这样一来，攻击实施操作容易且成本便宜。

但是，资源较少或经验较少的黑客仍在选择建立一个“广阔的网络”，以了解他们所捕捉到的猎物，尽管这种技术变得越来越不普遍，但也不是可以完全忽略的技术。这种方法通常用于确保“质量优先”。也就是说，可以针对许多人，但要求的赎金很低，人们通常会选择付款以确保文件的安全归还。

令人担忧的是，勒索软件攻击发展的下一阶段可能会开始破坏文件，而不仅仅是对其进行加密。这意味着即使公司选择支付要求的赎金，他们也可能无法重新获得对其宝贵的数据访问权，这将对被攻击者造成巨大的损失。

保护和预防

勒索软件预防需要分层次进行，仅仅使用一种安全工具，无论多么彻底，都将不足以完全保护企业的宝贵数据。需要安全团队对其进行从内到外的安全加固。

修补程序是安全的起点，因为它可以解决大多数致命问题，比如导致黑客进入网络和工厂勒索软件的软件漏洞。尽管良好的补丁程序管理有效，但由于时间和资源密集，该过程经常执行不当或执行不充分。安全团队正在使用自动化进行该过程，从而确保扫描所有的站点和服务器以查找丢失的补丁程序以及成功修补的已知关键漏洞。由于 BlueKeep 的出现，这一点就显得尤为重要。BlueKeep 是 Windows 操作系统的较早版本中的一个漏洞，有可能被利用，其攻击力远不及 WannaCry。

有时打补丁是远远不够的，例如，它不能抵御 0 day漏洞攻击，这就是为什么必须采用分层方法来实施安全策略的原因。在这些情况下，重要的是阻止无法通过应用程序白名单修补的内容。

除了通过修补已知漏洞来建立外部防御之外，确保内部源不授予勒索软件访问权限也至关重要。黑客的另一个常见切入点就是网络钓鱼攻击。这些电子邮件使用社会工程学的策略和人性的本能来欺骗企业的员工，使其点击恶意链接，从而将勒索软件嵌入电脑。无论员工身处哪个部门，都必须对员工进行网络安全教育，并且要教给他们有关可疑电子邮件的注意事项，无论他们懂不懂安全，安全意识都是应该具备的。

与防病毒保护和电子邮件过滤器配合使用时，教育和意识可以成为抵御勒索软件的另一种有效的防御层。想要培养安全意识可前往 二向箔安全 官网进行学习，最近推出了一系列免费的网络安全技能包，有关渗透测试、安全运维、CTF、黑客技巧尽在其中，双十一限时超多优惠。

从勒索软件中恢复

除预防措施外，企业还必须确保在最坏的情况下制定应急计划。除了制定数据恢复计划（例如确保定期存储备用备份）外，公司还需要决定是否响应赎金要求。显然，支付赎金会带来巨大的财产损失，但也可能造成业务损失，声誉受损或有价值的数据丢失等相关的成本。

考虑到谁的钱用来支付赎金，这种情况变得更加严重。政府和地方当局正在成为越来越多的网络攻击的目标，仅在2019年上半年，英国议会就经历了2.63亿次未遂攻击。如果政府机构决定支付赎金，付清网络犯罪分子的赎金所使用的钱是公众税。

在某些情况下，支付赎金在商业上是有意义的。例如，数据丢失或服务中断造成的抵制成本大于付款成本。但是，这不是我们所推荐采用的方法。相关部门指出，任何人都不应支付勒索的要求，因为这样做只会鼓励攻击者继续犯罪活动，并且也没有保证数据会真正的释放。坚定地说，如果一家企业已正确备份其文件，则可以无需付款并回到备份时的状态。

其实，最重要的是，如果公司做好了充分的准备和保护，他们就不必担心是否要支付勒索软件赎金，因为它们将无法渗透其内部应用数据。但是，勒索软件攻击正在不断发展，并且变得越来越有针对性和有效性。企业必须保持自己的网络安全状况，以领先于试图闯入的恶意攻击者。
