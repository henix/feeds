机器之心新年全球招聘：开发、记者、分析师与各类实习生

机器之心

专业的人工智能媒体和产业服务平台

机器之心是全球化的人工智能信息服务平台，一直为人工智能从业者提供高质量知识及商业信息，主营业务包括媒体、报告、数据库产品、活动和产品服务等。不久前，机器之心刚刚完成 A 轮融资，投资方包括星路资本、人民网和 TalkingData 等，这是机器之心的第三轮融资，此前还获得联想之星、今日头条、科大讯飞和第四范式等机构的天使轮融资的Pre-A轮融资。


机器之心将持续生产和运营高质量内容，并正在开发全新的信息产品，我们需要全球范围内更多的开发、内容、分析和运营人员加入。


高级编辑（全职 2 名）


工作职责：


负责栏目整体内容建设的执行与效果分析，系列专题的选题策划、组织推进，并配合平台达成内容团队的考核目标。


岗位要求：


  1. 拥有足够的信息敏感性、一定程度的产业洞察力、认识和解读行业命题的逻辑框架；

  2. 接受过专业的采写训练，具备专题策划能力；

  3. 在科技、商业类媒体工作 3 年以上，有管理记者的经验、熟悉人工智能行业是加分项。


资深记者（全职 2 名）


工作职责：


负责常规内容生产，挖掘和研究国内外人工智能领域优秀创业公司，并撰写相应的人物、公司和产业报道。


岗位要求：


  1. 对人工智能前沿技术及其产业应用有浓厚求知欲、好奇心；

  2. 具备基本商业常识，逻辑性强，能够多角度思考热点话题；

  3. 有良好的职业态度与团队协作精神，接受过专业采写训练，在科技、商业媒体工作 2 年以上。


编辑助理（1-2 名）（实习生岗）


工作职责：


  1. 编译、校对英文文章。

  2. 协助编辑撰写技术、产品、公司和行业相关文章。


岗位要求：


  1. 良好英语翻译能力；

  2. 对前沿技术感兴趣，有热情。

  3. 工作地点：北京、武汉


以上除编辑助理岗位皆为全职，工作地点在北京，有意者请将简历及三份作品发送至：hr@jiqizhixin.com 或 liuyan@jiqizhixin.com。



分析师（1 名）


工作职责：


负责人工智能领域的信息搜集、整理及分析，撰写行业简报和分析报告。


岗位要求：


  1. 拥有极强的信息检索、逻辑思维、分析和写作能力；

  2. 对人工智能的产业应用有深入了解，具备基本的人工智能技术知识；

  3. 优秀的英语阅读能力；

  4. 1-2 年独立的行业研究和分析经验。


全职，工作地点在北京，有意者请将简历及三份作品发送至：hr@jiqizhixin.com 或 zhaoyunfeng@jiqizhixin.com。


实习编译（2 名）


工作职责：


  1. 编译、校对英文文章；

  2. 撰写技术、产品、公司和行业相关文章；

  3. 跟踪国内外人工智能热点事件；


岗位要求：


  1. 卓越的英语翻译能力；

  2. 英语或计算机相关专业毕业者优先；

  3. 对前沿技术有一定了解和热情；

  4. 需保证每周至少三天到岗工作。


产业服务实习生（1 名）


工作职责：


  1. 整理技术应用案例；

  2. 参与行业调研；

  3. 支持产业活动。


岗位要求：


  1. 对人工智能技术应用及商业化有一定了解；

  2. 对参与产业活动，接触行业从业者感兴趣；

  3. 金融、管理、市场营销等相关专业毕业，有四大及咨询机构实习经验者优先；

  4. 需保证每周至少三天到岗工作。


新媒体运营实习生


岗位职责：


  1. 负责公司新媒体平台的日常内容运营（包括但不限于微信、微博、知乎、官网等）；

  2. 负责完成各渠道内容的编辑、优化和发布工作；

  3. 有良好的文案基础，能敏锐发掘热点，灵活撰写文案；

  4. 配合线下活动的运营和推广工作。


岗位要求：


  1. 大三或大四学生，一周保证 4 天以上工作时间；

  2. 性格开朗，保证良好的跨部门沟通交流，执行力强；

  3. 机器之心的长期读者优先，热爱人工智能行业优先。


以上实习岗位工作地点在北京，有意者请将简历发送至：hr@jiqizhixin.com。


ruby 开发工程师（1名）


基本要求：


  1. 优秀的逻辑，理解分析业务能力强；

  2. 拥有一年以上实际 ruby on rails 开发经验；

  3. 有基本的前端知识, 包括 HTML, CSS, Javascript 等；

  4. 良好的编码习惯，对代码的简洁和美有追求；

  5. 写测试并不把测试当做无意义的负担；

  6. 英文阅读以及快速学习能力，喜欢尝试新技术，认同工程师文化。


加分项：


  1. github 有高 star 项目；

  2. 掌握多种编程语言，熟悉函数式编程语言；

  3. 爬虫经验及 AI 相关技能；

  4. 区块链开发经验。


前端工程师（2名）


基本要求：


  1. 熟练掌握各种 Web 前端技术，包括但不限于（react/vuejs/graphql/appllo/webpack/es6/sass）；

  2. 代码规范，使用 ESLint ，写测试并不把测试当做无意义的负担；

  3. 有一定的前端架构能力，代码组织清晰。


加分项：


  1. 有一定的设计功底，对好的产品设计、视觉设计、交互设计有自己的见解；

  2. 有丰富的前端模块化的经验，维护过模块化的前端 UI 组件；

  3. 掌握至少一门后端语言, 熟悉 ruby on rails；

  4. 使用 react native 开发过 app 或有 PWA 项目经验。


以上岗位工作地点在北京，简历连同个人 github 地址一齐发送至 ligan@jiqizhixin.com，以上职位接受实习。
