衍盛中国

深圳总部
衍盛中国成立于2014年，是由前高盛执行董事、股票衍生品交易部负责人章友先生及其技术团队创立的专业从事量化投资的资产管理公司。公司投研和技术人员占比超过70%，核心成员来自海内外著名高校并曾在高盛、瑞信、瑞银、汇丰银行等享誉全球的金融机构任职，具备国际化视野和丰富的投资经验。公司在香港的投资平台持有香港证监会核准的“9号牌照”，能够挖掘更多元化的投资机会，满足客户多元化的投资需求。

北京清华大学实验室
衍盛-清华量化投资实验室成立于2016年9月，是由衍盛中国捐赠设立，隶属于清华经管中国金融研究中心的实验室。该实验室集金融研究、量化人才培养、商业应用为一体。实验室由衍盛中国策略团队核心成员亲自带队，是学生们获得市场实战经验的宝贵渠道，同时为同学们提供衍盛中国团队自主研发的策略研究和开发实践平台。

Python高级工程师

岗位职责：

  1. 数据清洗，整理和初步分析；

  2. 高频数据系统优化；

  3. 因子挖掘，协助交易团队设计模型；

  4. 机器学习模型的实现，迭代和改进。

任职条件：

  1. 计算机，电子，数学，物理，统计，金融等相关专业，985高校全日制本科及以上学历；

  2. 熟练使用python，熟练使用数据分析和机器学习相关的package；（如numpy，pandas，statsmodels，pytorch等）；

  3. 态度认真，踏实负责，学以致用，能自始至终完成一个完整的项目。

    具有以下条件之一或以上者优先考虑：


    基础扎实，有一定的C/C++编程能力，对技术充满热忱；

    数理信息技术奥赛，或者acm，mcm，kaggle等比赛获奖者；

    有相关行业的良好项目经历或者独特的见解。

C++开发工程师

岗位职责：

  1. 量化交易平台的开发及维护，包括但不限于行情对接、交易对接、订单管理、风控管理、仓位管理、性能优化等。

任职条件：

  1. 3年以上linux C++开发经验，计算机、电子、数学、统计、金融等相关专业，全日制本科以上学历；

  2. 掌握STL/Boost 等核心库、掌握多线程编程、掌握网络优化TCP/IP-UDP，熟悉TCP/IP协议，有socket编程经验；

  3. 熟悉网络IO、熟悉常用通信模型select、poll等、熟悉Linux内核、具有内核、驱动开发经验、熟悉高性能内存数据库相关技术；

  4. 理解数据结构和算法、理解面向对象编程；

  5. 具备量化交易系统开发经验者优先；

  6. 具备低延时、多进程通信的交易系统开发经验优先；

  7. 具备网络编程、操作系统、FPGA开发、大数据分析开发经验优先；

  8. 态度认真，踏实负责，学以致用，能自始至终完成一个完整的项目。

测试工程师

岗位职责：

  1. 参与版本上线的全流程工作，包括需求分析、制定测试发布计划并提交生产发布建议；

  2. 环境的搭建及部署，能够快速对测试环境进行搭建和部署，并根据要求准备对应测试数据；

  3. 测试执行，负责每个版本的测试质量保证工作;

  4. 持续集成，负责测试持续集成平台的管理及完善，自动化测试用例库的补充；

  5. 熟悉软件工程及项目管理方法。

任职条件：

  1. 计算机或相关专业本科以上学历， 1~3年金融行业测试经验；

  2. 熟悉常见的CS架构系统测试，熟悉测试流程及测试案例设计的基本方法；

  3. 有测试开发经验或自动化测试经验，熟悉常见的测试工具（如：selenium，QTP等）；

  4. 掌握Python/Java/JavaScript中至少一种语言,熟悉Linux/Unix操作系统，熟悉MySQL数据库操作；

  5. 沟通能力较好，计划条理性较强，细致认真；

  6. 有证券、基金行业实际的工作经验优先。

Python组实习生

岗位职责：

  1. 股票，期货，期权等衍生品交易数据分析，策略开发和实现等；

  2. 对不同金融数据库（wind，朝阳永续，bloomberg等）的使用，数据校对，分析和可视化等；

  3. 对实盘或者回测中产生的交易订单分析，为基金产品的交易和运营决策提供数据支持。

任职条件：

  1. 计算机，电子，数学，统计，金融等相关专业，全日制本科或硕士，不限于在校生；

  2. 有一定的编程基础，熟练使用office；

  3. 每周需要至少有三天或者五个半天及以上的全职工作时间；

  4. 态度认真，踏实负责，学以致用，能自始至终完成一个完整的项目。具有以下条件之一或以上者优先考虑：


    熟练使用python，有较强的python编程能力和一定的python优化经验；

    高中大学相关数学，编程竞赛（包括不限于acm等比赛），数学或者cs基础良好；

    具有一定金融知识，有过相关国内金融衍生品交易或者做过金融类数据分析相关项目经验；

    即将毕业或实习时间在三个月及以上。

量化研究实习生

岗位职责：

  1. 研究股票、期货、基金以及各种衍生品的中高频交易策略模型，按照研究目的协助完成模型的实现、回测与优化；

  2. 协助完成量化交易程序的开发、测试、优化与维护。

任职条件：

  1. 理工科背景的本科或以上学历，如数学，统计，计算机，物理，金融工程等专业；

  2. 扎实的c++语言基础，较高的C++编程能力；

  3. 熟练使用linux、sql、python者为佳；

  4. 品学兼优，善良、诚实、有责任心，具有良好的沟通能力和团队合作精神；

  5. 具备很强的自学能力，具有相关实习经验者优先。

基金运营实习生

岗位职责：

  1. 协助对接外部合作机构；

  2. 协助投资人尽调工作；

  3. 协助基金产品成立运作、账户开立等工作;

  4. 协助基金信息披露工作；

  5. 协助完成领导交办的其他临时性工作。

  1. 经济、金融、财务等经济管理类相关专业；

  2. 应届毕业生和在校学生均可；

  3. 熟悉基金、证券基础知识；

  4. 做事积极主动，有较强的沟通协调能力和责任心；
