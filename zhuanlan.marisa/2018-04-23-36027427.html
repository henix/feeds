原文

安装方式（两种）：

1.进入终端，输入下面的命令：

raco pkg install DrRacketTheme

在不同操作系统上可能有所不同，如果没有成功，请尝试先进入racket安装目录下然后在运行这条命令。

2.打开DrRacket，打开File菜单下的Package Manager，切换到Available from Catalog选项卡，点击Update Package List按钮，稍等片刻，然后在Filter文本框中输入DrRacketTheme，点击Install按钮。

设置的时候请点击View菜单下的Set Background，选择需要作为背景的图片。

Set Background Alignment能够设置图片的对齐方式。

希望这款插件能够帮助到正在学习racket或是scheme的人。
