Quiz 17:

下面这段代码输出什么？

fn main() { let mut a = 5; let mut b = 3; print!("{}", a-- - --b); }

输出结果: 2

解读

考察要点：

  1. 和Quiz16代码一样，Rust中不存在自增或自减运算符

a-- - --b等价于a - (-(-(-(-b)))) ，5 - 3 = 2。

那为什么Rust中没有自增/自减运算符呢？

在Rust官方的FAQ中有描述：

先增和后增（以及相对应的减法）虽然方便，但也相当复杂。它们都需要求值顺序的知识，而且经常导致 C 和 C++ 中的细节错误和未定义的行为。 x = x + 1 或 x += 1 只是略长一点，但更加明确。
