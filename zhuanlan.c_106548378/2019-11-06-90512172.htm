比如有1、5、8、12这几种纸币，现在有n元，想要兑换，最少需要多少张纸币？

--------------------------我是罪恶的分割线-------------------------

其实一看就知道是动态规划：

不难写出程序：

coin <- c(1,5,8,12)
n <- 100000
num <- 1:n
num[coin] <- 1
for (i in 1:n) {
  if (num[i]==1) next
  for (x in coin)
    if (i>x) num[i] <- min(num[i-x]+1, num[i])
}

因此不大难。有点像走楼梯的题目。
