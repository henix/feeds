xray不仅是漏洞检测工具和poc编写、运行工具，更是一个开放性的技术交流平台，是安全技术爱好者的大本营。高级版便是在原有社区版的基础上，充分吸纳使用者的反馈建议后，应运而生。

相比于社区版，高级版都增加了哪些功能呢？小姐姐提前来点剧透，比如：Struts 2、ThinkPHP系列的检测插件、子域名收集、Web爬虫等等，其他更多功能敬请期待！！

先来了解一下这四个功能吧。

Struts 2

从2007年至今，已披漏的Struts相关漏洞数量近60个，而Struts 2作为世界上最流行的Java Web服务器框架之一，在金融、互联网等行业被大规模使用。xray作为精细化漏洞检测工具的代表，有针对性的处理了多种框架类应用。

xray安全研究团队对Struts 2漏洞的检测逻辑进行了深入研究，与其他漏洞检测工具相比，xray高级版请求数量更少、检测更加精准，安全指数更高，支持检测Struts2的OGNL表达式注入（涉及到s2-001、s2-013、s2-048、s2-053、s2-012、s2-014），devMode代码执行，s2-005、s2-015、s2-016、s2-032、s2-037、s2-045、s2-052、s2-057等高危漏洞。

Thinkphp

Thinkphp是一个诞生十多年的老牌框架，在中国互联网圈里的使用也极为广泛，很多站长以及平台都在使用这套开源系统来建站。xray支持参数识别和处理后调用通用sql注入模块来检测Thinkphp的sql注入漏洞，同时还支持 `InvokeFunction` 和 `method` 系列远程代码执行漏洞的无害化检测。

子域名收集

xray高级版子域名收集主要分为三个部分：DNS域传送检测、外部信息源收集子域名（持续增加并且随版本维护信息源）和子域名爆破。

特色功能：

1. 可选可视化交互Console UI，支持一键复制结果，一键保存文件；

2. 域名泛解析自动检查与智能规避泛解析；

3. 子域名递归爆破，层数可控；

4. 支持高并发（DNS查询并发），且并发数量可精准控制：在200的 DNS并发查询下，几十秒内可以完成对常见域名（非爆破）的检测；

5. 内置主字典3000余条，副字典160+条（用户检测多级域名）。

Web爬虫

在xray 0.8.0版本中，我们加入了以爬虫方式触发的主动扫描，受到了很多社区朋友的认可，也收集了一些不足之处，因此我们计划投入核心精力加强Web 爬虫。

为了更适应专业用户的使用，为了打造专业的被动扫描引擎，为了打造专业的Web 爬虫，我们将合并现有社区版爬虫扫描与企业版中的部分智能化特性，与扫描引擎解耦，在高级版中作为独立的爬虫功能提供。这样我们可以方便的只使用 Web 爬虫进行业务资产清点，而不进行漏洞扫描，也可以给 Web 爬虫加代理实现原有的爬虫扫描功能。
