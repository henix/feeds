原文

Shodan 搜索引擎大家一定都不陌生吧，一个“搜天搜地搜万物”的搜索引擎，互联网上的一切都逃不过它的魔爪，什么打印机、电脑、手机都在它的掌握之中。

但是，使用 Shodan 需要注册，以及 API 才能调用。

下面的这款工具，可以让你摆脱这种烦恼，无需注册获取 API，也能使用这么强大的搜索引擎！

Metabigor

让你可以通过命令行使用搜索引擎（例如：Shodan、Censys、Fofa等）进行查询，而无需任何 API 密钥。这简直就是黑客们的福音啊，洞察互联网连接设备的世界。

项目地址：https://github.com/j3ssie/metabigor

优势

  • 不需要使用 API 密钥，因此不必担心 API 引用的限制
  • 可以从没有管理员权限的命令行中进行查询
  • 无需 root 帐户即可获得更多结果
  • 但是我有一个高级帐户，为什么我需要这个狗屎？
    • 使用 API 能查到的，它都可以
    • 查询将得到优化，与手动搜索或使用 API​​ 密钥相比，可以获得更多的结果
    • 搜索结果不会重复

如何运作？

Metabigor 可以使用你的 cookie 来模拟来自浏览器的搜索并优化查询以获取更多结果。

支持的搜索引擎：

  1. Shodan
  2. Censys
  3. Fofa Pro

安装

拉取项目源码：

git clone https://github.com/j3ssie/Metabigor

安装依赖：

pip3 install -r requirements.txt

基本使用：

./metabigor.py -s <source> -q '<your_query>' [options]

完成后会将 IP 地址列表都很好地存储在一个文本文件中，且每行只有一个。

更多选项可查看帮助菜单：

./metabigor.py -h
