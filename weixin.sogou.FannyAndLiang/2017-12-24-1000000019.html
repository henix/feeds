健芬与周导

FannyAndLiang

这里是谢健芬与周亮的自留地。


自从去年年底定下来一年要读50本书这个目标以来，内心很是焦虑。焦虑有三方面，第一是相中一本书后买纸质书还是用Kindle读，Kindle电子书的好处是去食堂路上可以读、吃饭可以读、吃完蹲厕所还可以读，但电子书总是给人一种虚无缥缈的感受；第二是读完一本后接下来读什么，这时候往往陷入迷茫中，虽然我有一份书单，但是往往是起兴时加进来，一段时间过后又觉得无趣味，所以书单也并不能解决问题；第三是读完了感觉不写点东西特别容易忘，但是写点东西又很费时，起初是以“阅读量并不大，等有点积累再说”来安慰自己，慢慢地觉得读写始终是要合一才能更有效率，况且一年之后也实在毫无脸面去这么讲了。


但好在功夫不负有心人，全年进入读完统计的书共计53本，不包括一些读到一半便丢弃不顾的书以及一些编程技术类并没有完整读完的书。数量在其次，算是一个好的开头，因为毕竟人生的前三十年从未将读书纳入到重要的事务范畴。数量在其次，毕竟其中有精华当然也有糟粕，每次读到糟粕就痛心不已，但读到精华也兴奋不已，渐渐也找到了识别之法。数量在其次，很多书读完即走，走掉即忘，但无论如何从中找到了读书的乐趣，也许这是最大的收获了。


数量在其次，重点是反思。第一，所读的书中猎奇为主，所以并没有什么主题可言，这样对很多问题的分析和思考并不深刻，因此未来需要围绕一定的主题，去刨根问底。第二，大部头并不多，读了很多历史演义类的书籍，对历史人物和故事是有所了解，但更深层次的理解和感悟并不多。第三，读完即忘，留存笔记或心得几乎没有，所以未来计划每周都要产出一篇心得，虽然目前文笔比较欠缺，但不练习就不会有进步。


再列举一下所读的书中的经典之作，值得反复阅读的：


文学类


黄金时代

王小波 / 湖南文艺出版社

王小波的比喻很有其独到的角度。读书的时候感觉坐老司机的车，猛地急转弯，一下子没反应过来，好慌，等反应过来觉得很过瘾，需要强忍住才能不爽得笑出来，因为在外人看起来看书还笑很傻。但是又实在忍不住，还是要笑。于是在外人看来就变成了看书想笑但强行忍住不过又忍不住最终还是笑了出来，反而更傻了。《我的阴阳两界》很治愈，《革命时代的爱情》有点啰嗦，可能要花时间慢慢看才行。总之，此书必须一本放到书架上。 


文化苦旅

余秋雨 / 长江文艺出版社

高中读过，前些月Kindle读了一遍，前几天买一本收藏，再读一遍。纸张很好，打开一阵纸香居然盖过墨香，吸引得我赶紧埋头凑近去闻去嗅，可见作者与编辑之只用心。 


解忧杂货店

[日] 东野圭吾 / 南海出版公司

看完了《岛上书店》再看《解忧杂货铺》，感觉后者比前者不知道高到哪里去了 。



经济管理


富甲美国

 山姆·沃尔顿 / 江苏文艺出版社

对拿第一的信念的坚持，对节省每一分钱信念的坚持，对零售业的无比喜爱，再加上沃尔顿卓越的领导能力造就了沃尔玛的奇迹。读完后查了一下沃尔玛2015财年的交易额为4821亿美元。


零售心理战

铃木敏文 / 江苏凤凰文艺出版社


零售的哲学：7-Eleven便利店创始人自述

[日] 铃木敏文 / 江苏文艺出版社

这两本都是日本7-Eleven创始人铃木敏文的作品，但是《零售的心理战》比前《零售的哲学》要更好，《零售的哲学》容量有限但想说的太多，说得不透彻。销售力就是让消费者觉得来对了，买对了，吃对了的能力。不要为消费者着想，要站在消费者的立场想。


腾讯传

吴晓波 / 浙江大学出版社

吴晓波写得很好，作为经历了互联网崛起的一代，对一些故事很有感触。其中的一些腾讯做产品的故事提现了腾讯做产品的方法，非常值得借鉴。后面几章可能是要交稿，有点马虎，但是瑕不掩瑜。


管理思想史（第六版）

丹尼尔·A·雷恩 / 中国人民大学出版社

这是一本经典，管理理论入门。


创新者的窘境

克莱顿•克里斯坦森 / 中信出版社

不仅仅是每个企业家都应该必备的书，也是每个大众应该了解的书，当你读了这本书，再看产品跌宕、企业兴衰就很明白了。另外，弱肉强食适者生存的时代，对于每个大众而言，如何能让自己始终跟随潮流，不落伍，也应该读读这本书。


一本书读懂财报

肖星 / 浙江大学出版社

非常清楚明白，一个老师讲解一个课题，就应该像这本书讲解得一样。读完还需要在心理仔细琢磨，这是一个需要像大学学习知识一样熟读、背诵、能好好应用的科目。


管理的未来

加里.哈默尔 / 中信出版社

很有感触，其实这都已经不是未来了。这本书居然没有实体书卖，只能读Kindle版。


华杉讲透孙子兵法

华杉 / 江苏文艺出版社

古文的书籍读起来很费劲，因此我一直对这类书籍是近而远之。但是这本书有点超乎我的意料，作者讲得很清楚，有其他家的注解也有自己的解释，对于入门非常合适。但是其中有些观点难免感觉太过于发挥，但仍然非常值得一读。



心理


登天的感觉

[中国]岳晓东 / 安徽人民出版社

这本书有点老了，但是并不过时，因为心理科学本身进展比较缓慢、心理问题至今仍然很难解决。这本书令我惊奇的地方在于作者对案例的反思能力，心理的发展其实正是来源于自我的反思，其实这种反思能力在任何一个行业都适用。



科普


复杂

梅拉妮·米歇尔 / 湖南科学技术出版社

人工智能非常火，这本书其实是非常好的人工智能入门书籍，真正告诉你人工智能要解决什么问题，如何解决这些问题，以及当前的进展是如何的。



历史


曾国藩大传

林乾 / 人民文学出版社

这是一本正儿八经的历史书籍，并不是小说演义类，两者的区别在于前者更多地从人物的语言、文章、事迹来分析人物的心理过程，而后者则绝大部分篇幅是讲故事。所以要读曾国藩，一定要读这本。


红星照耀中国

[美] 埃德加·斯诺埃 / 人民文学出版社

力荐。很多事物并不像他自己所说的那么千好万好，但是也并不妨害多去了解他。就像是两家创业公司的创业故事，其中一家在某个时间点能胜出就说明有其道理，这个道理是值得了解和借鉴的。小故事挺有趣，真的不要再嘲笑湖南人h发成f音了，这个笑话已经80年了，该让它过去了[捂脸] 


毛泽东传

[美] 罗斯·特里尔 / 中国人民大学出版社

分析非常细致的一本书，读完之后对毛泽东有了全新的认识。



宗教与哲学


西方哲学史

[美]梯利 / 商务印书馆

哲学入门经典，非常难啃，但是要入门就得啃。


圣经的故事

[美]房龙 著 张稷 译 / 中华书局

读这本书是因为对哲学有兴趣，但是要了解哲学就先得了解西方历史，要了解西方历史，就不得不了解基督教的历史，这本书很好的解答的了我的问题。很繁琐，但是跳着看没问题的。



最后


以上是我读完后认为值得一读的书，做记录以供参考。至于明年的计划，目前只是确定依然是50本书，但书单还没有定。另外还有个目标，给老婆读5本书。
