原文

两三个月前，和知乎上认识的facebook工程师覃超（就是我到了湾区后，雪中送炭，借我奔驰小跑的朋友）一起吃了顿饭，同时认识了美女Daisy，特赞（tezign）的cofounder。当时Daisy递我的一张精致方块名片让我耳目一新，聊下来大概了解了特赞是个设计师的freelancer marketplace（自由职业者市场）。


说起自由职业者市场，不得不提odesk。互联网最擅长的一件事，就是构建平台，解决市场两端的信任问题，同时突破时空的限制，利用高效透明的信息传递，让本来难以繁荣的一个个分散的小市场变成一个大一统的市场。劳务派遣（staffing）市场在美国是个千亿美金规模的市场，而线上的自由职业市场才仅仅到达十亿的量级 [1]，尚有很大的空间。odesk通过切下和互联网开发相关 [2] 的蛋糕，把开发者和企业对接起来，成就了一个坚实的商业模式。去年底elance-odesk [3] 获得了三千万美金的融资 [4]，走上了上市的康庄大道。


来到湾区后，微信上和Daisy还偶有联系，同时耳边时不时都能传来特赞的名字。由于我是搞网络安全的，得知去年安全领域炙手可热的初创公司trustlook（真格基金投资）利用特赞这个平台对接了设计师，不由得对这个名字更加侧目。和Daisy详聊之下，才算对特赞有了一个全方位的了解。


和odesk类似，特赞也在自由职业市场这块大切糕上切细分市场。只不过，由于创始团队的独特气质，他们切的是设计行业中线上可交付的部分：交互设计，平面设计（VI，名片，logo），插画设计（图片，海报），意图打造一个平台，对接企业的设计任务和设计师，打造一个有品质的设计师自由职业者市场。


感谢苹果对设计和美的孜孜以求，如今在移动互联网大潮之下，无论是创业公司，还是大中型企业，都意识到设计的重要性 —— 产品要通过精美的设计和流畅的交互让用户情不自禁地wow出来。这种"wow"难以名状，就是当你第一次使用google map，第一次打开instagram，第一次遇见pinterest，第一次升级path 2.0后那种用起来很爽，有那么些相见恨晚的感觉。而为了抓住这种感觉，公司们对设计的重视程度越来越高，需求越来越多。


市场的强劲需求及团队的独特气质就是特赞这个产品的想法能够迅速得到投资人青睐并获得不错的天使融资的原因，其投资人有线性资本（王淮 - facebook早期员工），还有华创资本（熊伟铭 - 之前在途客圈和他打过交道）等。


「团队的独特气质」是一件事A团队能做好，B团队却不见得能做好的一个重要因素。做一个像特赞这样的marketplace，早期的设计师人脉很重要。特赞的创始人范凌先后毕业于同济，普林斯顿（建筑系硕士），哈佛（设计博士），曾任教央美和伯克利，在国内外的设计圈有诸多人脉，而特赞的团队也有浓厚的央美背景，这也是特赞还在内测阶段就已经对接了一些高质量的项目的原因。


由于特赞的网站还处在封闭内测阶段，我也无法体验更多的功能，跟他们的研发的lead朱彬（曾经是硅谷一个创业公司的CTO）聊了聊，后端主要是使用 java & scala / play framework / mysql & mongodb / redis / solr，前端是backbone，整个系统的最激动人心也是最挑战的点是：客户给出一个设计项目，如何尽可能少人工干预的情况下，通过算法分析，找到和客户需求最匹配的设计师？


这也是我觉得特赞和odesk不同的地方。odesk也有设计相关的部分，客户从几十上百满足条件的设计师中需要花力气挑选（甚至面试）合适自己的设计师，因为对方水准的良莠不齐，整个过程漫长而充满挫败感；而特赞把这部分麻烦揽到了平台本身，为客户挑选合适的设计师，保证有品质的设计。


这在技术上是一个挑战。我还不知道特赞会如何去做。在音乐领域，通过算法已经能够建立情感模型，用户对已有歌曲的喜好会成为系统为其推荐类似情感模型的歌曲的准则。视觉领域，也应该会有类似的模型。切容我狂想一下，特赞会让用户通过像hunch [5] 那样，做些有趣的视觉选择题，来获取客户的视觉偏好，然后，从设计师上传的作品集中选择与用户偏好最贴切的。当然，这想法也许要累积足够多的数据才做得起来，或者说，做得有意义。起初，也许就是神奇的标签匹配（标签也是一门学问哦），甚至是人工匹配 —— 系统总是在发展中不断完善的嘛。
