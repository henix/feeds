Jaeles

这是一个用 Go 语言编写的 Web 应用程序扫描器，它功能强大、灵活易扩展，堪称用于自动 Web 应用程序渗透测试的瑞士军刀。

项目地址：https://github.com/jaeles-project/jaeles

结构框架

其原理大致为，通过浏览器或者命令行工具使用 jaeles 扫描器，获取Urls列表或原始请求，并与签名结合以生成请求。发送这些请求并运行检测脚本以确定请求是否易受攻击。

安装

下载

下载地址：jaeles-project/jaeles

Go 环境 版本>=1.13，运行以下命令。

GO111MODULE=on go get -u github.com/jaeles-project/jaeles

使用

命令：

jaeles scan -s 'jira' -s 'ruby' -u target.com

＃扫描具有多个输入的
jaeles scan -s signatures/common/phpdebug.yaml -U /tmp/list_of_urls.txt

＃被动扫描多个签名
jaeles scan -v --passive -s "signatures/cves/jira-*" -u http://example.com --params 'oob=xxx.burpcollaboraborator.net'

jaeles scan -c 50 -s 'java' -x 'tomcat' -U list_of_urls.txt

jaeles scan -c 50 -t 3 -s '/tmp/custom-signature/.*' -U list_of_urls.txt

cat urls.txt | grep 'interesting' | jaeles scan -c 50 -t 3 -s 'fuzz/.*' -U list_of_urls.txt --proxy http://127.0.0.1:8080

jaeles server --verbose -s sqli

初始化签名和 Web UI

命令：

jaeles config -a init

将新签名加载到数据库

只需将新签名保存到任何地方，然后通过以下命令加载即可。

jaeles config -a reload -F /tmp/custom-signatures/

/tmp/custom-signatures/ 自定义签名文件夹

或者，也可以通过以下命令直接使用新签名

jaeles scan -s new_sign.yaml -u example.com

扫描

Scan list of URLs based on signatures

Usage:
  jaeles scan [flags]

Flags:
  -h, --help          help for scan
  -s, --sign string   Provide custom header seperate by ','
      --ssrf string   Fill your BurpCollab or any Out of Band host
  -u, --url string    URL of target
  -U, --urls string   URLs file of target

BURP 插件安装

获取 JWT

第一次运行 Jaeles 时，它将在 /.jaeles/config.yaml 中设置一些配置和凭据。使用该凭据获取 JWT。

如何安装Burp插件

下载：https://github.com/jaeles-project/jaeles-plugins/blob/master/jaeles-burp.py

通过使用此命令 jaeles server -v，将 Jaeles 作为服务器模式运行。首次运行时，它将在〜/ .jaeles/burp.json 上创建默认凭据。

Jython 安装在 Burp 上，然后加载它。Burp-> Extender->添加 jaeles-burp.py 作为 python 扩展。

加载插件后，它将自动登录并将持久性JWT存储在 〜/.jaeles/burp.json 中。
