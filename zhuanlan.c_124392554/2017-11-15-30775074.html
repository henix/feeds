某天在“知乎”看到了这个问题，然而之后搜不到了= =。

提问者要求使用尽可能简单的模型和尽量少的数学假设。

那我就在“2+2=4”、“加法交换律、结合律与数学归纳法”这两篇文章基础上谈谈我的理解：

回答“为什么(-1)×(-1)=1”，首先需要厘清两个问题：

  1. 什么是乘法？
  2. 什么是“-1”？

比较容易接受的处理的方式是——从皮亚诺自然数公理化定义出发不断定义和扩充，虽然它很繁琐（由于过长，以下只介绍基本路线而不给出详细严谨的定义和证明）：

➊ 自然数中的乘法可以如下递归定义：

  • a \times 1 =a ，
  • a \times S (b) = a\times b + a 。

可以证明它满足：

➀ 对加法的分配率 a \times (b+c) = a\times b + a \times c （固定 a,b 对 c 做归纳）；

➁ 结合率 (a\times b)\times c=a\times(b\times c) （固定 a,b 对 c 做归纳）；

➂ 1\times a=a （使用归纳法）；

➃ 交换律 a\times b=b\times a （固定 a 对 b 做归纳，并利用➂和➀）。

➋ 由于之前的介绍中。“自然数”是从1开始的，因此需要补充定义一个新事物——0。

定义一个新的对象“ 0 ”，并对所有自然数 a 定义 0+_1a=a+_10=a 、 0+_10=0 ，然后将原有的“ + ”运算和新的运算“ +_1 ”合起来还称作“ + ”，并可以证明它满足：

➀ 交换律（这很自然）；

➁ 结合律——对任意自然数 a,b 证明 0+(a+b)=(0+a)+b ， a+(0+b)=(a+0)+b ， a+(b+0)=(a+b)+0 。

➂ 还可以证明“ 0 ”的唯一性即若 a+0_2=0_2+a=a ，则 0_2=0 。

并可以记 S(0)=1 。

再定义新的运算“ \times_1 ”为 a\times_1 0=0 ，然后和原有的“ \times ”合并还称为“ \times ”，并可证明它仍然满足➊的➀➁➂➃。

➌ 然后引入一类新事物（负数），对自然数 a 定义“ -a ”，并定义运算“ +_3 ”， a+_3(-a)=(-a)+_3a=0 。

然后在新事物内部定义运算“ +_4 ”：(-a) +(- (b)) = - ( a + b ) 。

再之后定义就是自然数和负数之间的加法“ +_5 ”（论述就太麻烦了，而且后面用不到）。

最后把“ + ” 、“ +_3 ”、“ +_4 ”、“ +_5 ”合在一起，构成“ + ”，然后可以证明它内部无矛盾，满足交换律、结合率，还可以证明“相反数的唯一性”，即若 x+a=a+x=0 ，则 x=-a 。由此还可以得到 -0=0 。

➍ 之后就是将原来自然数上的乘法扩充到 a\times (-b) ，到现在大意已经明了：就是要扩充后还保持原有的➊的➀➁➂➃，那么由于 0=a\times 0=a\times(b+(-b))=a\times b+a\times (-b) ，及“相反数的唯一性”，只能定义为 a\times (-b)=-(a\times b) 。

➎ 然后 0\times (-b)=-(0\times b)= (-0)=0 。

➏ 最后就是将原来自然数上的乘法扩充到 (-a)\times (-b) ，仍然要保持原有的➊的➀➁➂➃，那么 0=((-a)+a)\times (-b)+a\times (-b)=(-a)\times (-b)+a\times (-b)=(-a)\times (-b)+ (-ab)

于是 (-a)\times (-b)=-(-(a\times b)) ，由“相反数的唯一性”，只能定义为 (-a)\times (-b)=a\times b 。

好了剩下的 (-1)\times(-1) 就不是问题了。

前面我是挣扎着写下去的😱（虽然很多地方也都跳过了），下面介绍一种不是很直观的方式，而且涉及到“等价”的概念。

➊ 如前定义自然数之间的乘法。

➋ 引入自然数组成的有序二元组 (a,b) ，其中 a,b 都是自然数。

定义这样的二元组之间的等价性：若 a+d=b+c ，则称二元组 (a,b) 与 (c,d) ，将 (a,b) 所在的等价类记为 [(a,b)] （就是表示通常所说的“ a-b ”），可以证明这是一个等价关系。

在二元组之间可以定义“ + ”运算： (a,b)+(c,d)=(a+b,c+d) 。之后可以验证若 (a_1,b_1) 与 (a_2,b_2) 等价、 (c_1,d_1) 与 (c_2,d_2) 等价，则 (a_1,b_1)+(c_1,d_1) 与 (a_2,b_2)+(c_2,d_2) 等价。

于是可以将这个运算扩展到等价类上： [(a,b)]+[(c,d)]=[(a+b,c+d)] 。

利用自然数加法满足的各种性质，可以证明这个运算满足交换律、结合律。

定义二元组之间的“ \times ”运算： (a,b)\times(c,d)=(a\times c+b\times d,a\times d+b\times c) 。可以类似地证明可以将这个运算扩展到等价类上—— [(a,b)]\times[(c,d)]=[(a\times c+b\times d,a\times d+b\times c)] 。之后利用自然数乘法满足的各种性质，也可以证明这个运算满足交换律、结合律、对等价类的加法有分配律等性质。

➌ 所有 [(S(a),1)] 构成的集合和自然数集合同构：

  • [(S(a),1)] 对应于 a ；
  • [(S(a),1)]+[(S(b),1)] 对应于 a+b ；
  • [(S(a),1)]\times[(S(b),1)]=[(S(a\times b),1)]（推算过程很繁琐）对应与 a\times b 。

➍ 用记号 0 表示 [(1,1)] ，则可以证明对于任意自然数 a ， [(S(a),1)]+[(1,1)]=[(1,1)]+[(S(a),1)]=[(S(a),1)] 。

➎ 对于任意自然数 a ，用符号“ -a ”表示 [(1,S(a))] ，则可以得到

[(S(a),1)]+[(1,S(a))]=[(1,S(a))]+[(S(a),1)]=[(1,1)] 。

（事实上这就对应于通常说的“ a 的相反数 -a ”）

➏ 于是

  • [(S(a),1)]\times[(1,1)]=[(1,1)]\times[(S(a),1)]=[(S(a)+1,S(a)+1)]=[(1,1)] ;
  • [(1,S(a))]\times[(1,1)]=[(1,1)]\times[(1,S(a))]=[(1+S(a),1+S(a))]=[(1,1)] ;
  • [(S(a),1)]\times[(1,S(b))]=[(S(a)+S(b),S(a)\times S(b)+1)]=[(1,S(a\times b))] （就是“ a\times (-b)=-(a\times b) ”）；
  • [(1,S(a))]\times[(1,S(b))]=[(1+S(a)\times S(b),S(a)+S(b))]=[(S(a\times b,1))] （就是“ (-a)\times (-b)=a\times b ”，特别是 (-1)\times(-1)=1\times1=1 ）。

而 \{[(S(a),1)]|a是自然数\}\cup\{[(1,1)]\}\cup\{[1,S(a)]|a是自然数\} 就称作整数集合。

事实上，整数就是所有自然数的二元组在这个“等价”关系下的等价类——虽然这个结果对于上述讨论并不重要。换言之对于任一个自然数的二元组 (a,b)，若 a=b 则它与 (1,1) 等价，否则必定存在自然数 c 使得或者它与 (S(c),1) 等价或者它与 (1,S(c)) 等价。

这一点的证明需要用到如下结果（自然数的三岐性）——对于任意两个自然数 a,b 则下述三者有且仅有一者成立：

  1. a=b ；
  2. 存在自然数 c 使得 a+c=b （此时称“ a<b ”）；
  3. 存在自然数 c 使得 a=b+c（此时称“ a>b ”） 。

（然后，我也懒得给出这个结果的证明了。思路是固定 a ，然后对 b 做归纳；过程中必要时对是否 a=1 、是否 c=1 做讨论。）

于是对于二元组 (a,b) 而言，下述三者有且仅有一者成立：

  1. a=b ；
  2. 存在自然数 c 使得 (a,b) 等价于 (1,S(c)) ；
  3. 存在自然数 c 使得 (a,b) 等价于 (S(c),1) 。
