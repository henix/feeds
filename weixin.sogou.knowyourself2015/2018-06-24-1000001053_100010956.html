今天，「KY成长计划」开启第5期招募，这是面向青年学生的实习计划。


名为“成长”，我们是希望大家能在KY倡导并践行的工作氛围中，体会到从学生到职场人的转变：感知自己肩上的责任，体会自己的行为对他人的影响。


我们希望按照KY信奉并一直在努力践行的认真、诚实、创新的工作准则，结合我们赋权、平等、自由、多元的价值观，为大家营造一个能够在实习工作中找到真实自我、并参与创造社会价值的创业环境。


因此邀请更多和我们一样，受过（某方面的）专业的训练、喜欢创新和不断迎接挑战、有意愿加入“提升中国社会的心理幸福程度”这份事业的小伙伴，成为我们并肩的战友。


「KY成长计划」具体情况：


实习地点：上海市长宁区，地铁直达写字楼


适宜人群：

1. 2019届毕业生，本科、硕士均可；

2. 可保证每周（周一至周五）全职工作最少4天，实习期三个月及以上 。

实习薪酬：120-150/天


表现优秀者有转正机会哦，来投简历吧！

以下是在招实习岗位：


· 内容中心 ·


内容实习生（3人）


工作内容：

1. 参与KY内容在知乎、今日头条、微博、豆瓣等平台上的改编创作和分发；

2. 参与知乎问答、悟空问答、感情问答等不同平台和不同主题内容有关的问题答复；

3. 参与心理专业科普内容的收集整理，比如实验、研究等；

4. 参与人文社科美学领域的素材收集，比如经典文选、名画名曲；

5. 参与热点事件的响应和创作。


任职要求：

1. 熟悉KnowYourself的文章；

2. 拥有本科及以上学历，其中心理学、社会学或社会工作专业优先考虑；

3. 平日里有使用知乎、今日头条、豆瓣、微博等平台的习惯，对互联网新近发生的事情心里有数，网感好；

4. 具备良好的中文写作能力，逻辑性好，英文阅读能力较好；行文清晰准确、简洁；

5. 耐心，仔细，认真，负责，抗压能力强，有团队合作精神；

会摄影，会作图，有策划活动的思维是加分项。



心理测量实习生（2人）

R&D Intern – Psychometrics

 

工作内容：

1.针对大众感兴趣的心理、行为科学相关的研究、信息及报告等进行整合、解读；

2.根据需求，配合心理测试或量表的研发，包括但不限于概念细化，数据处理等；

3.优化测试或量表的语言文字，呈现形式等

 

任职要求：

1. 重点大学心理学、社会学等相关专业，有“心理量表”开发经验优先；

2. 具备心理测量问卷/量表开发的基本素养，掌握定量数据分析的技能；

3. 有很好的脑洞及文字能力，能够将研究量表的语言生活化、趣味化；

4. 熟练使用SPSS，Excel等软件；

5. 主动性强，有极强的学习能力与归纳总结能力，熟悉互联网。



研发实习生（2人）

R&D Intern – Psychoeducation


工作内容：

1. 针对大众感兴趣的心理、教育、知识付费等领域的行业信息及报告等进行整合、解读；

2. 根据需求，配合完成相应内容产品的策划与研发；

3. 根据受众与内容产品的供需情况，优化该产品的语言文字，呈现形式等。

 

任职要求：

1. 重点大学心理学、社会学等相关专业，有项目经验的优先；

2. 具备询证科学的基本素养，掌握定性与定量数据分析的技能；

3. 有很好的脑洞及文字能力，能够将研究量表的语言生活化、趣味化；

4. 熟练使用SPSS，Excel等软件；

5. 主动性强，有极强的学习能力与归纳总结能力，熟悉互联网。



· 市场中心 ·


市场实习生（5人）


工作内容：

1. 协助部门内对外市场合作对接，公司内外资源的沟通，把控项目进度；

2. 参与到市场活动的创意策划、落地执行和宣传推广中；


任职要求：

1. 快速学习能力强，有网感，脑洞大，责任心强，沟通力强；

2. 热爱心理学、对自媒体行业有了解。



视频实习生（3人）


工作内容：

1. 负责短视频前期策划及文案脚本等工作；

2. 负责视频拍摄、后期剪辑和素材搜集整理；

3. 能进行简单的后期处理；

4. 负责视频栏目运营相关工作，包括内容上传，合作认证等；

5. 视频竞品分析，以及每周数据整理分析。


任职要求：

1. 熟练使用FCP、PR等剪辑软件，熟悉AE、AU等软件优先；

2. 掌握摄像技术和应用技巧者优先；

3. 熟悉流行短视频应用、二次元文化优先；

4. 做事认真负责，善于学习，沟通能力强，善于协调工作。



音频策划实习生（2人）


工作内容： 

1. 负责广播音频自制类节目，挑选选题，策划节目形式，文案改写，跟进外包制作，以及剪辑和上线推广节目，能够把握一档节目的全案策划运营；

2. 逻辑思维好，文案能力强，选题上手快； 

3. 对互联网用户的需求有敏感度； 

4. 想法多、团队合作意识强、沟通能力好，能承受一定的工作压力，不拖延；


任职要求：

1. 熟练使用Audition、Premiere等软件；

2. 有广播、电视台、视频网站、制作公司实习经历者优先；

3. 新闻、广播电视编导及录音、播音主持等专业方向优先； 

4. 思维活跃、脑洞大、有阅读储备和良好的学习能力者优先。



· 产品中心 ·


产品实习生（4人）


工作内容：

1. 负责市场需求收集、参与用户调研，做好竞品分析；

2. 协助产品经理，进行产品规划、业务流程设计、功能设计及相关文档撰写； 

3. 协助跟进运营需求，在产品负责人的带领下设计产品迭代需求；

4. 与研发、运营等团队进行沟通确保对产品的理解，协助推进开发，跟进测试和上线。


任职要求:

1. 具有良好的交流沟通能力，敏感的数据意识；

2. 能够使用Axure/Visio软件；

3. 有良好的逻辑思维与分析能力，学习理解能力强；

4. 具备数据收集分析，文档撰写能力。



· 运营中心 ·


运营实习生（3人）


工作内容：

1. 群运营：协助维护运营KY各类粉丝群；

2. 课程跟进：协助开课，维持课堂、学习群秩序，整理课件等学习资料；

3. 微信体验优化：图文排版、创新形式研发、关键词设置等；

4. 运营互动活动创意策划：监测其他平台优秀活动，策划我们的活动；

5. 客服工作：协助回复用户问题，安抚用户情绪；

6. 数据统计：熟练使用Excel表进行数据统计。


任职要求：

1. 心理学及社科背景优先；

2. 脑洞能力强，有创意有想法，并且有把事情执行出来的动力和能力；

3. 工作认真仔细负责，有耐心沉得住气，能从琐碎事情中体会工作乐趣；

4. 会微信排版(及各类排版工具)、PS基本操作，会HTML5语言或者简单编程者加分；

5. 有较强的沟通能力，英语较好。



· 销售中心 ·


客户执行实习生（2人）


工作内容：

1. 负责广告项目的执行，包括客户对接、内部协调、流程管控、供应商管理等；

2. 负责搜集整理行业及竞品动态，并制作分析报告；

3. 负责支持销售中心其他工作。


任职要求：

1. 有较强的责任心，具备快速学习成长能力；

2. 工作踏实，执行能力强，能多线程管理工作，条理清晰不出错；

3. 熟练掌握Microsoft Office、PS等办公软件，能高效制作各类文档；

4. 个性开朗，善解人意，擅长沟通。



· 总裁办 ·


人事实习生（3人）


工作内容：

1. 协助考勤管理，员工加班与假期登记，每月打卡记录处理 ；

2. 协助面试接待，办理员工入职离职续签手续和前期材料准备； 

3. 负责实习生招聘、入职、离职、考勤相关事宜 ；

4. 资料收集、归档、整理、发放等。


任职要求：

1. 工作认真仔细负责，具备Office基本技能，PS熟练使用；

2. 有较强的沟通能力，逻辑能力强，英语较好。
