so，VMware 登场（“网管”终于可以休息了），开启虚拟机的时代，从 win2003 开始，到无数个 win7 被玩坏；从 MS08-067 开始，到 CVE-2018 进行漏洞复现。

VMware虚拟机：通过软件模拟的具有完整硬件系统功能的、运行在一个完全隔离环境中的完整计算机系统。在实体计算机中能够完成的工作在虚拟机中都能够实现。在计算机中创建虚拟机时，需要将实体机的部分硬盘和内存容量作为虚拟机的硬盘和内存容量。每个虚拟机都有独立的CMOS、硬盘和操作系统，可以像使用实体机一样对虚拟机进行操作。

就这样，我们认识了 kali（神器），也是“杀害”众多虚拟机的“凶手”。

Kali Linux：一个基于 Debian 的 Linux 发行版，它与其它的 Linux 系统最大的不同就是它预装了13大类（如下图），300多个安全测试和渗透软件，包括大名鼎鼎的nmap(端口扫描器)，Wireshark(数据包分析器)，Aircrack-ng(一应用于对无线局域网进行渗透测试的软件)，John the Ripper(密码破解器)等等等等，Kali Linux预装的这些软件基本包括了黑客会使用到的所有工具。

对于这种简单的寝室局域网，根本没放在眼里，一个接一个寝室搞，从 wifi 入手进入他们局域网，再发起各种攻击，随后就是各种游戏断线、卡机等等，感觉特有成就感。

arpspoof -i eth0 -t 192.168.0.106 192.168.0.1

开始了解 wireshark，获取数据包，分析，了解 tcp/ip 协议，学习各种包头、结构。

Wireshark：一个网络封包分析软件。网络封包分析软件的功能是撷取网络封包，并尽可能显示出最为详细的网络封包资料。Wireshark使用WinPCAP作为接口，直接与网卡进行数据报文交换。

Burpsuite：用于攻击web 应用程序的集成平台，包含了许多工具。Burp Suite为这些工具设计了许多接口，以加快攻击应用程序的过程。所有工具都共享一个请求，并能处理对应的HTTP 消息、持久性、认证、代理、日志、警报。

参加 CTF（虽然没很高的名次，但认识了很多大佬，也学到很多东西）。

CTF：中文一般译作夺旗赛，在网络安全领域中指的是网络安全技术人员之间进行技术竞技的一种比赛形式。CTF 起源于1996年 DEFCON 全球黑客大会，以代替之前黑客们通过互相发起真实攻击进行技术比拼的方式。

在 CTF 这段时间，隐写、加密、web、代码审计、逆向、内网渗透等，充满了每一天的生活，简单的一张图片竟藏有如此大的秘密。（winhex、二维码、binwalk、RGB、摩丝电报等等）

当然，有些时候网上的工具也不是万能的，或者说有时候并不是那么的方便，对于一个会敲代码的网工同学来说，这并不会难倒我们，我们开始写一些小工具、小脚本，比如下面这个 base64 转换器👇

# 导入 base64模块
import base64

def fun1(str1):
    # 需要转成2进制格式才可以转换，所以我们这里再手动转换一下
    result = base64.b64encode(str1.encode())
    result=str(result,'utf8')
    # 打印转换后的结果
    return result

def fun2(str1):
    # 再把加密后的结果解码
    result = base64.b64decode(str1)
    # 同样的，解码后的结0果是二进制，我们再转换一下
    return result.decode()

while 1 :
    n = input("1.base64加密 2.base64解密 3.退出\n")
    if(n=='1'):
        str01 = input("请输入字符串：")# 给定需要转换的字符串
        print(fun1(str01))
    elif(n=='2'):
        str02 = input("请输入字符串：")# 给定需要转换的字符串
        print(fun2(str02))
    elif(n=='3'):
        break
    else:
        break
