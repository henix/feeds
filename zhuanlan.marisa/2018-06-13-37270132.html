PCP Theorem的定义


我们先给出一个传统的证明系统 (proof system)的定义：

  • 我们有一个命题 (statement)
  • 有一个证明者 (prover) 会给出一个证明对上面的命题证明或证否
  • 验证者(verifier)检查上述证明，然后接受或拒绝这个证明


The PCP (Probabilistically Checkable Proof) Theorem:

对于任何一个在NP里面的语言 L ，我们都有一个如下的P.C.P. 系统

  • 对于一个长为n的输入
  • 然后证明者给出一个多项式长度的证明
  • 验证者生成 O(\log n) 长度的随机串，根据这个随机串在证明里面随机的选 C 个位置。这里的 C 是一个常数。
  • 验证者读取这 C 个位置的内容，然后进行测试来决定接受或拒绝这个证明。

这里的验证者要满足如下条件:

  1. 可靠性：对于任何不满足条件的输入，验证者接受它的概率不超过一半
  2. 完备性：对于如何满足条件的输入，验证者一定接受它

PCP Theorem和 \textrm{GAP-3SAT} \in \textrm{NP-hard} 等价

\textrm{GAP-3SAT} _{c, s}(0<s\leq c\leq 1) 的定义：给一个有m个子句(clause)的3SAT的表达式，假设我们最多能满足其中的 k 个句子。

  • 如果 k\geq cm ，接受这个表达式
  • 如果 k < sm ，拒绝这个表达式
  • 如果 sm \leq k < cm ，我们不在意这种情况，随机选择


定理1. PCP Theorem和 \textrm{GAP-3SAT}_{1,s} \in \textrm{NP-hard} 是等价的

证明:

\Leftarrow 假设 \textrm{GAP-3SAT}_{1,s} \in \textrm{NP-hard}。我们这里会证明3COLOR有P.C.P.系统。假设输入是一个有 n 个点的图G。那么验证者首先把这个输入归于到一个有m个子句的 \textrm{GAP-3SAT}_{1,s} 表达式 \phi 。这个时候证明者会被要求给出一个 \phi 的赋值。然后此时证明者，只需要随机选择一个子句，获得对应的三个布尔变量的赋值就好了。这个时候显然 C=3 。

可靠性 (Soundness)：如果G不能被三染色的话，那么我们有 \phi 满足的子句小于 sm 。那么我们随机选择的子句不满足的概率是 s 。如果这个时候 s>\frac{1}{2} 的话，我们重复 O(1) 次就好了。

完备性 (Completeness)：如果G可以被三染色的话，那么\phi 一定可以全部被满足。所以我们接受的概率是1。

\Rightarrow 假设我们有PCP theorem，这个时候我们把3COLOR规约到 \textrm{GAP-3SAT}_{1,s} 来证明 \textrm{GAP-3SAT}_{1,s} 是NP-hard的。首先我们有PCP theorem，那么我们可以得到一个多项式长度的证明，在这里我们会认为证明是对于一个3SAT表达式的赋值。然后，我们用 O(\log n) 长度的随机串生成 2^{O(\log n)}=\text{poly}(n)=N 个随机选择。对于每个这样的选择，我们会生成 C个位置，然后读取这 C个位置的值，把他们标作 x_1,x_2,\ldots,x_C 。我们定义一个布尔函数 H(x_1,x_2,\ldots,x_C) ，当原有的3SAT表达式能在不改变这C个位置的值满足时返回一，否则返回0。因为Cook-Levin Theorem，我们知道我们可以在多项式的时间内生成这个函数对应的3SAT表达式。因为只有 C 个变量，我们不妨假设这个对应3SAT表达式有 K=2^C\cdot C 个子句。然后，我们用 \land 把这 N 个长度为 K 的表达式连接在一起，得到了一个有 N\cdot K 的3SAT表达式 \phi 。

完备性 (Completeness)：如果原有的输入是可以三染色的话，那么我们一定可以得到完美的证明，或者说一个满足所有子句的赋值。那么我们新的3SAT表达式显然是可以满足所有子句的。

可靠性 (Soundness)：如果原有的输入不是三染色的话，那么根据PCP theorem，我们每次至少有一半的机会得到一个不可满足的3SAT表达式。那么这个3SAT表达式不可满足的时候，它最多有 K-1=K(1-\frac{1}{K}) 个可以满足的子句。那么对于 \phi ，我们能够同时满足的不超过 \frac{N}{2}K(1-\frac{1}{K})+\frac{N}{2}K=NK(1-\frac{1}{2K})=m(1-\frac{1}{2K}) 。也就是说我们能够得到一个常数 s=1-\frac{1}{2K} 。


我们后面将通过证明 \textrm{GAP-3SAT}_{1,s} \in \textrm{NP-hard} 来证明PCP theorem。
