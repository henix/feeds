阻击华为 美国做梦

2G、3G移动通信技术主要由欧美推动，中国显然不想再错过有望再度改变互联网和其他传统行业的5G时代，于是中美之间正上演一场主导权争夺战，华为因为技术优势被盯上了。


华为全球首席财务官(CFO)孟晚舟在加拿大被捕的消息让中美两国之间的“科技对抗”再一次暴露在聚光灯之下。尤其是近期澳大利亚、新西兰、英国等国家相继禁止华为参与当地的5G建设。中国在5G建设上投入了大量的时间金钱和人力，从某些方面来说，中国是领先的，美国方面的研发活动只由几个重要的科技公司主导。

毫无疑问，华为在全球5G布局上是数一数二的领军企业，自2009年启动5G技术研发以来，已经投入了超过10亿美元。据《华尔街日报》报道，一个由政府牵头的委员会与中国多家移动运营商和设备生产商在5G的测试和研发上展开合作。在国家牵头和庞大内需市场护航下，华为等中国公司得以销售大量5G设备，并在这个过程中积累宝贵经验。

美国在此方面感到了压力，开始使用各种“工具手段”，动用各个盟国帮助其遏制华为。《金融时报》援引英国情报人员的说法，华为正是因为技术优势而被盯上。美国政府的行为可能是为了让华为“减速”，以便市场的其他企业能够赶上。

根据咨询公司德勤(Deloitte)8月进行的一项研究，中国自2015年以来花费已比美国多出240亿美元，已建造了约35万个蜂窝基站，而美国不到3万个。德勤还指出，中国每一万人有14.1个基站，美国则为4.7个。这对5G至关重要，因为5G网络需要的蜂窝基站数量远远多于4G。

《华尔街日报》援引美国富瑞金融集团(Jefferies LLC)驻香港电信分析师Edison Lee的话表示，对中国而言，5G是一个如此重要的战略项目，涵盖方方面面，动用所有资源。他说，因为中国若跨入5G大门，那也会在6G、7G、8G上占得先机。

德意志银行经济学家张智威评论此次孟晚舟事件称，科技领域已沦为新战场。国泰君安团队在点评中美贸易战停火时称，未来已经不再仅仅是 “关税战”，2019年的重点是知识产权战、科技战。

//
领先一步
//


在5G技术的支持下，无论是无人驾驶汽车上路行驶，还是医生进行远程复杂手术，都有可能变为现实。该网络可以为所谓物联网、虚拟现实和增强现实中的联网设备提供依托。信号塔将把高速互联网接入设备，降低设备对线路和Wi-Fi的依赖。

哪个公司首先掌握了5G技术，未来的回报将是巨大的，尤其是在知识产权方面，据专家称，拥有专利的公司将获得数十亿美元专利使用费。

从国家层面上讲，拥有最大和最可靠网络的国家将在研发需要更快网速支持的技术方面领先一步。占主导地位的设备供应商可以让国家情报机构和军方在监视或扰乱敌对国家网络方面拥有优势。

中国引领5G未来的雄心很大程度上取决于能否制定技术标准，全球其他地区必须遵守这些标准，并支付使用和许可费用。为此中国一直在制定全球标准的国际电信产业团体中扮演积极角色。

据《华尔街日报》称，华为聘请了现已倒闭的设备制造商北电网络有限公司的前高级研究员童文，童文无意中发现了“极性编码(polar coding)”，这是土耳其科学家Arikan发明的修正数据传输错误的新方法。

2016年在内华达州里诺Peppermill举行的一次关键的标准制定会议上，华为与一家西方公司进行了激烈的竞争，会议以一种折中的方式结束，该标准的一部分将采用极性编码，这将赋予华为一项关键专利的所有权。

//
美国奋起直追
//


美国国家科学基金会在2015年发表了一份关于5G的内部报告，其中显示，相较于欧洲、韩国、日本和中国，美国在这一领域非常落后。

德勤主要负责人之一Dan Littmann在一项声明中说，“美国想保有竞争力而且最终取得领导地位的话，就应谨慎评估5G之争并且迅速采取行动。”美国也开始准备集中力量办大事。

根据路透今年1月份报道，据两名国会助手表示，出于国家安全考虑，美国议员敦促国内第二大无线运营商AT&T切断与中国手机厂商华为的所有商业联系，并反对中国电信运营商中国移动进入美国市场。

美方称是基于国家安全考虑，并相继组织了一系列中国收购交易，包括蚂蚁金服收购速汇金的交易。

9月份美国联邦通讯委员会(简称FCC)本月早些时候宣布，计划不理会涉及小型蜂窝发射器部署的一些地方性法规和费用，以加快5G网络的建设。小型蜂窝发射器是5G网络的一个重要组成部分。NSF正协调各方，为5G以及未来几代无线网络构建测试平台。

贝尔实验室前研究员、NSF董事Thyaga Nandagopal是该平台项目的领导人，企业、学术机构和政府部门可以在该平台上协同测试5G及其他无线网络应用。在纽约，哥伦比亚大学等学术机构运营着一个由NSF资助的试验场。

Nandagopal称，NSF的努力侧重2020年后的无线研发，而不是5G部署的最初几年。“我们可以进行战略性投资，并获得比其他人更好的结果。”

《华尔街日报》报道称，美国一些电信公司正在抢占屋顶和电信杆，用于部署可加快网速的小型基站，同时也在施压设备制造商生产5G兼容产品。

//
为什么是华为？
//


一方面加快发展，一方面遏制对手，这是美国的策略，而华为等科技公司就成为斗争的对象。华为今年9月中旬宣布已经在全球签署了22份5G商业合同。

全球最大的网络设备制造商华为自2012年以来一直被禁止进入美国，华盛顿方面称这些公司的设备可能被用来监视美国人。

据CNBC报道，华为轮值主席徐直军称：“有一个问题一直困扰着我。美国政府作为一个国家权威，为什么他们一直针对华为这样的商业公司？是因为我们在5G上太好了还是有什么不为人知的理由？说5G会带来网络安全问题，我会说这些言论是出于政治动机而根本不是基于事实。所有这些怀疑都没有任何经实质证据”。

特朗普政府今年3月份阻止了总部位于新加坡的博通收购美国芯片巨头、5G领军企业高通公司，理由是担心博通会削减高通的研发资金，并使中国企业在5G领域处于领先地位。

中国也给予了反击，7月份否决了高通收购荷兰芯片制造商恩智浦半导体的交易，这宗交易本可帮助高通从联网汽车等新市场的5G投资中获利。

后来美国就采取明目张胆、拉帮结派的方式参与竞争了。

美国的盟友澳大利亚8月份以对间谍活动和安全的担忧为由，禁止华为竞标5G网络的建设。华为当时给澳大利亚政府的公开信中表示，此举是“过于片面”的，并且“没有事实依据”。《纽约时报》指出，华为在担心其他大客户，如英国、加拿大、德国、新西兰，会效仿美国。

这其中的英国、加拿大、新西兰恰好和美国、澳大利亚共同组成了所谓的五眼联盟（Five Eyes），即英美协定下组成的国际情报分享团体。华为的担忧是不无道理的，这些国家都在陆续的开始行动。

美国还用财务援助来力劝盟友远离华为。知情人士称，对于德国、意大利和日本等广泛使用华为设备的盟友，美国官员已将其所认为存在的网络安全风险告知这些国家的官员和电信高管。其中一些人士称，美国还考虑为那些避开使用中国设备的国家增加电信开发方面的财务援助。

“对于华为而言，作为5G技术的领导者，我们没有机会通过5G解决方案和服务为美国消费者提供服务，那么美国市场就是一个没有完全竞争的市场，同时仍然阻碍了主要参与者的参与。现在，我不确定他们是否能真正实现成为世界第一的5G目标。”徐直军表示。

作为第一批签署“一带一路”倡议的国家之一，南太平洋国家巴布亚新几内亚近日表示，尽管遭到澳大利亚、日本和美国的反对，华为仍将为其完成5G网络的建设。

该国国家投资部长William Duma表示，巴布亚新几内亚没有敌人，无论澳大利亚或美国对网络安全看法如何，作为发展中国家，不会愚蠢到拒绝其他国家在电信网络建设上的帮助，不会放弃建设一半的项目。（完）
