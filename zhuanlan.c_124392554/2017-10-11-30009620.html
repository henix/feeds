有一个球形的面团，有一位和面高手和和，有一位不会和面的面面。
面面把球水平转了一下（下图左）。
和和：你这样只转一下是不行的，中间轴不动，面没有和开。
面面接着又把球垂直转了一下（下图右）。
和和：你这样还是不行，球心不动，面仍然是没有和开。

面面：那你只要不扯断面团，不将面团粘连，而且最终还把面团成同样大小的球体；那么，一定还至少有一个点还在原来位置！所以你也没有把面和开！
和和：∑（〟〇О〇）真…真的吗!?
面面：是的！因为——

布劳威尔不动点定理（Brouwer's fixed-point theorem）

n 维欧氏空间中，紧致凸集上映到自身的连续映射 f 至少有一个不动点。即，存在一个点 x_0 ，使得 f(x_0)=x_0 。

等等，什么是“凸集”？

（简单地说）在\mathbb{R}^n 中，如果集合中的任意两个点相连所得到的线段(上的每个点)也在该集合内，则称该集合为凸集（convex set）。即若 x,y\in A 则对于任意 0\leq\alpha\leq1 有 \alpha x+(1-\alpha)y\in A 。
例如，二维平面中圆形和矩形都是凸集，但环形和马蹄形都不是凸集。

那什么是“紧致”？

“紧致"就是所有开覆盖都存在有限子覆盖，所以得先要解释——

什么是“开集”？

由于讨论的是 \mathbb{R}^n 中的凸集，所以这里只介绍 \mathbb{R}^n 中的开集：

设 A 是 \mathbb{R}^n 的一个子集。如果 A 中的每一个点 P=(x_1,x_2,..,x_n) 都有一个以该点为球心的小球（等同于P的邻域neighbourhood）包含于 A ，即存在 \delta>0 使得\{(y_1,y_2,..,y_n)\in\mathbb{R}^n| (x_1-y_1)^2+...+(x_n-y_n)^2<\delta^2 \}\subseteq A，则称 A 是 \mathbb{R}^n 中的一个开集（open set）。

例如：在 \mathbb{R} 中，开区间都是开集；开集都可以表示成可数个（这个概念以后再解释吧……||- -）两两不相交的开区间的并集；闭区间都不是开集；半开半闭区间都不是开集。

继续解释“紧致”

设 X\in\mathbb{R}^n ，若 C 是开集的集合且满足 {\displaystyle X=\bigcup _{x\in C}x} ，则称C 是 X 的一个开覆盖（open cover）。如果 X 的每一个开覆盖 C 都存在一个有限子集 C_1\subseteq C使得 {\displaystyle X=\bigcup _{x\in C_1}x} ，则称 X 是紧致的（compact）， C_1 称作 C 的有限子覆盖。


举个例子吧，实数集中的开区间 (0,1) 就不是紧致的： \{(\frac{1}{n},1-\frac{1}{n})|n\geq3\} 构成一个覆盖，但是不存在有限覆盖。

事实上，有——

Heine-Borel定理： \mathbb{R }^n 中集合 X 是紧致的，当且仅当 X 是有界闭集。

（集合 A 是闭集是指 \mathbb{R}^n -A是开集 ；

集合 A 有界是指存在 r>0 使得任意 (x_1,x_2,..,x_n)\in A都有 x_1^2+x_2^2+...+x_n^2<r^2 ）

回到布劳威尔不动点定理及和面问题： \mathbb{R}^3 中的球体是一个紧致凸集，所以如果把满足“不扯断面团，不将面团粘连，而且最终还把面团成同样大小的球体”要求的“和面”行为，那么它就是一个球体映到自身的连续映射，因此一定存在不动点——也就是没有和开的面（虽然只是一个抽象的点）。


布劳威尔不动点定理和巴拿赫压缩映射不动点定理是“不动点定理”们世界中最基本和著名的两个，前者的证明是“存在性”的而后者的证明是“构造性”的。

最后用“牛刀”宰只“小鸡”

众所周知，小明同学又被赶出课堂了。所以他决定去登山，上午8点从山脚出发，一路上走走停停，中午12点到达山顶。在山上赏月过夜后，第二天8点小明从山顶出发，原路返回下山，悠哉悠哉，走走停停（反正上课去也得被赶出来），中午12点恰好到达山脚。
那么必定存在某个时刻，小明两天中这时刻在山上的位置恰好一致的；或者说，两次到达山上某个地点的时间是相同的。

可以将这条路视作一个闭区间 [a,b] ，之后建立区间上到自身的连续映射——

f(上山时，时刻t所在地点)=(下山时，时刻t所在地点)

于是根据布劳威尔不动点定理 f 存在不动点，即 (上山时，时刻t_0所在地点)=(下山时，时刻t_0所在地点) 。


好吧，其实这只“鸡”也可以随便“宰一宰”——

假设下山的小明穿越回了前一天，于是8点时两个小明分别从山顶和山脚同时出发，12点时分别到了山脚和山顶，由于走得都是同一条路，所以他们一定会相遇。这就表明两次到达山上某个地点的时间是相同的。
