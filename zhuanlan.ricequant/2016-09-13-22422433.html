原文

Ricequant 正在招聘软件工程师加入我们来帮助改变金融世界。我们在搭建亚太地区第一个和最棒的基于浏览器的算法交易平台。我们正在扩大我们的团队来支持我们快速增长的用户和研发我们非常有趣、领先的金融科技（ Fin-Tech ）产品。


我们被几家著名的投资方投资，资金充足。


我们解决一些有趣的问题，比如：

  • 在我们的云端服务器的安全容器中运行任意的用户代码

  • 设计和搭建非常易于使用和强大的量化金融研究和开发工具以及 API

  • 设计数据模型和存储来构建真实世界的金融数据，并且不断优化来解决运行模拟交易时候的大吞吐、低延迟问题

  • 处理实时海量行情数据更新的高吞吐和低延迟问题

  • 监控和自动扩充、缩小我们的云端资源

  • 在 PC 端和移动端设计和搭建令人惊叹的用户体验

目前，我们使用 Java 、Python 以及 C/C++ 在青云和私有云上搭建了 Ricequant 。 我们也大量使用了一些流行的 NoSQL 数据库和私有的时间序列数据库，并且我们也在使用诸如 Docker 和 lxc 的容器技术。我们希望用尖端的技术来改变中国乃至亚太地区的量化甚至一些其他的金融科技产品。我们也打算做一些开源框架的尝试。


我们团队中有很多顶尖的工程师，公司位于深圳。来自于顶尖交易系统公司 ORC 、投行、腾讯、 baidu 和国内外著名高校的毕业生，我们成长的很快，我们还是一个快速成长的创业公司，对各个级别的职位都开放，甚至也欢迎优秀的刚毕业的新人加入：）金融背景完全不是必须（不过您有金融背景我们还是很欢迎的）


我们仍旧是一个小公司，因此你仍旧将可以对公司的发展轨迹产生明显的影响和贡献：我们相信雇佣最好的工程师并且给他们有趣、有挑战的问题去解决并且给予足够的自主权去解决这些问题。投递您的简历，请发送至： hr@ricequant.com
