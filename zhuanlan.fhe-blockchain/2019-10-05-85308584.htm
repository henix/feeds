编者注：国内计算机专业的名称是：计算机科学与技术；相关专业还有：软件工程。本文是国外作者，讨论的专业是：计算机科学（CS）。不过讨论的过程涉及到编程，工程以及系统分析。非常类似于国内曾经的软件水平考试，其中分为：初级程序员，程序员，高级程序员，系统分析师。


在本文的剩余部分，将讨论成为一名编码人员，程序员，最后是计算机科学家所需的技术技能。


编码员阶段


1- 编程


成为编码员的第一步也是唯一的一步就是学习编程。


这是您CS职业生涯中最简单的一步，它可以让您快速反馈您是否应该从事CS职业生涯。


在这个阶段，重要的不是特定的编程语言，而是您将要学习的概念。这些概念几乎适用于任何其他编程语言。


当你成为一名经验丰富的程序员时，你学习一门新的编程语言时，将不会超过一周的时间，所以不要浪费你的时间试图找到“完美”的编程语言，因为：

a）它不存在，b）无所谓。


话虽如此，我个人建议您从以下两种语言开始。我将解释这些选择背后的原因，你可以选择你最认可的方式进行学习。


  • Python

强烈建议你一开始学Python

因为Python是一种非常容易学习的语言。真的，真的很容易！

它是一种非常高级的语言，允许您只用几行代码编写真正的程序。

因此，在很短的时间内，您就将能够开发重要的项目。

要学习python，我强烈推荐《Python Crash Course》。（如下图，确保你获得更新的第二版）





我发现它对初学者非常有用。

而且这本书是基于项目的，所以你将在学习编码的过程中获得乐趣。


  • Java

为什么另一种语言是Java？原因是：因为它会教你一些Python中甚至不存在的编程概念。

例如，Python是一种动态类型语言，而Java是一种静态类型语言。如果您不知道这意味着什么，将在学习这两种语言后理解它。

Python和Java的结合是一种非常好的开始，因为它们一起为您提供了几乎任何其他编程语言所需的编程概念。


为了增加上述优势，Python和Java都在工业中大量使用。因此，您不仅要花时间学习为您进一步发展铺平道路的基础，还要学习一些非常有用且需求量很大的实用语言。


多年前我从《Java core》系列中学习了Java(如下图)。





提供两本独立的书籍。一个是Java基础，另一个是高级Java功能。

建议暂时不要使用高级功能。关注这一阶段的基本功能。

恭喜！现在你是一个编码人员了！


程序员


2- 软件堆栈


好。现在你可以编写而且可以做一些非常酷的代码东西了，但是你真的知道发生了什么吗？

假设您编写了一个非常简单的程序，只需添加两个整数并将结果打印到屏幕上。

在Python中，它看起来像这样：

x = 5

y = 10

print(x + y)

我认为你了解你的代码。运行代码的计算机应输出15。

但是你真的了解底层发生了什么吗？

变量赋值（x = 5）在硬件级别意味着什么？什么是x，数字5在硬件中如何表示？加法操作如何实际发生？结果如何最终出现在我的屏幕上？


在一天的工作结束之后，摆放在那里的计算机只是硬件芯片和电线的集合。

计算机如何真正理解你的代码？并完美地执行它？

事实是，你的代码只是冰山一角。您的代码下面还有很多其他层。它们共同使整个工作按照你期望的方式工作。


一个合格的程序员需要能够揭开这个谜题。

在这个级别，你需要从代码开始一直了解堆栈的所有层，一直到硬件层。

由Noam Nisan和Shimon Schocken所著的《The Elements of Computing Systems》(如下图)是我最建议你了解堆栈的每一层的基本信息的一本不错的书。





本书涵盖了基本级别的硬件，编译器，链接器和操作系统，使其非常适合初学者。

它将引导您完成创建第一个编程语言，为其创建编译器和链接器，然后创建操作系统的步骤。


3- 算法和数据结构


现在你处于一个非常好的状态，可以重新开始编程，但这一次有着完全不同的心态。

因为现在，你真的知道底层发生了什么。

你了解硬件最终将如何运行您的代码。

你知道您拥有有限的硬件资源，并且您了解有效利用可用资源的价值。

研究算法和数据结构将教会你如何以一种使代码更高效的方式编写代码，但是你可以定义效率。它可以是速度，资源利用率，或两者兼而有之。

你将在此阶段学习的技能是将普通编码员与程序员分开的一些主要区别。


事实上，谷歌，Facebook和亚马逊等大多数大型科技公司在面试过程中都非常关注数据结构问题。

在算法和数据结构方面，关于涵盖该主题的最佳书籍并没有太多争议。

算法导论（AKA CLRS）(如下图)。





请注意，数据结构和算法的内容是与描述语言无关的，因此与你使用的编程语言无关紧要。

但是，有些人更喜欢阅读特定于他们首选语言的书籍。

你可以找到许多优秀的语言特定的数据结构书籍，比如这个用于Java，而那个用于Python。


4- 网络


你的代码在独立的单个计算机上运行的情况非常罕见。

最有用的通信代码要么是与本地网络计算机，要么是与互联网中的其他计算机进行通信。


因此程序员需要有一个非常坚实的计算机网络工作基础。

在我看来，当我还是一名高年级本科生时，我遇到了最好的网络书。

《计算机网络：自顶向下的方法》(如下图)，超级容易理解，涵盖了你需要知道的所有网络基础。
