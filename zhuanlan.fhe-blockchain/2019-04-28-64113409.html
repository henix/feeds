原文

加密货币价值突然飙升和下跌的原因是什么？不同山寨币的市场是不可分割地联系在一起还是在很大程度上是独立的？我们如何预测接下来会发生什么？


关于加密货币的文章，比如比特币和以太坊，现在充斥着各种猜测，数百名自称为专家的人都在倡导他们期望出现的趋势。这些分析缺少的是统计数据的坚实基础。


本文的目标是使用Python提供对加密货币分析的简单介绍。我们将通过一个简单的Python脚本来检索，分析和可视化不同加密货币的数据。在此过程中，我们将发现这些具有波动的市场行为以及它们如何发展的有趣趋势。




第1步 - 设置数据实验室

本教程旨在供各级技能的爱好者，工程师和数据科学家使用。您需要的唯一技能是对Python的基本了解以及设置项目的命令行的足够知识。


步骤1.1 - 安装Anaconda从头开始安装此项目的依赖项的最简单方法是使用Anaconda，一个预先打包的Python数据科学生态系统和依赖项管理器。

要设置Anaconda，我建议按照官方安装说明进行操作 - https://www.continuum.io/downloads。

如果您是高级用户，并且您不想使用Anaconda，那就完全没问题; 我假设你不需要帮助安装所需的依赖项。随意跳到第2部分。


步骤1.2 - 设置Anaconda项目环境安装Anaconda之后，我们将要创建一个新环境以保持依赖关系的有序性。

运行conda create --name cryptocurrency-analysis python = 3为我们的项目创建一个新的Anaconda环境。

接下来，运行source activate cryptocurrency-analysis（在Linux / macOS上）或激活加密货币分析（在Windows上）以激活此环境。

最后，运行conda install numpy pandas nb_conda jupyter plotly quandl以在环境中安装所需的依赖项。

这可能需要几分钟才能完成。

如果您计划在计算机上开发多个Python项目，那么将依赖项（软件库和包）分开以避免冲突是有帮助的。

Anaconda将为每个项目的依赖项创建一个特殊的环境目录，以保持所有内容的组织和分离。


步骤1.3 - 启动一个Interative Jupyter Notebook一旦设置了环境和依赖项，运行jupyter notebook启动iPython内核，然后打开浏览器访问http：// localhost：8888 /。

创建一个新的Python Notebook，确保使用Python [conda env：cryptocurrency-analysis]内核。


步骤1.4 - 在Notebook顶部导入依赖关系

一旦打开空白的Jupyter Notebook，我们要做的第一件事就是导入所需的依赖关系。
