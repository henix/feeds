第二行一个正整数 M ，接下来一行有 M 个数，表示 S_1 中包含的元素

第四行一个正整数 K ，接下来一行有 K 个数，表示 S_2 中包含的元素

样例输入:

5
3
1 3 4
3
2 3 5

2 5

题解:

这也能算题么....用Python的话，set有一个函数difference()，可以直接解决问题....

_ = int(input()) M = int(input()) S1 = set([int(x) for x in input().split()]) K = int(input()) S2 = set([int(x) for x in input().split()]) ans = sorted(S2.difference(S1)) if ans: print(*ans) else: print('None')

#include <iostream> using namespace std; int main(){ ios::sync_with_stdio(false); int N;cin>>N;int K; while(N--){ cin>>K; int rem = K / 3; int h = rem / 60; int m = rem % 60; if(m > 0) h += 1; int s = 1 - h + 24; int e = 0 - m + 60; if(s >= 24) s -= 24; if(e >= 60) e -= 60; if(s < 10) cout<<0<<s<<":"; else cout<<s<<":"; if(e < 10) cout<<0<<e<<"\n"; else cout<<e<<"\n"; } return 0; }

2. サンタ服(圣诞老人服)

题意：输入一个立方体的长，宽，高和操作次数。对于每一次操作， d_i=0表示横着切(沿着X切)， d_i =1表示竖着(沿着Y切)。求切完之后最小的一块的体积。

这个题比之前就要复杂一点点，但是也不是非常难：只需要对两种操作排个序，然后把横着间距最小和纵着切间距最小的长度记下来，最后乘以Z即可。

X, Y, Z, N = [int(x) for x in input().split()] op = [[0, X], [0, Y]] # 先把整个块看作切出来的.. for i in range(N): di, ai = [int(x) for x in input().split()] op[di].append(ai) ans = [X, Y] for i in range(2): op[i].sort() for j in range(1, len(op[i])): ans[i] = min(ans[i], op[i][j] - op[i][j-1]) # 贪心 print(ans[0] * ans[1] * Z)

题意：输入正整数 N ，求 N 的阶乘的最右9位非零数是什么，如果这9位有前缀0，那么去掉前缀0输出。简单来说就是：求一个阶乘去掉后缀0之后的后9位去掉前缀0之后的结果(有点拗口2333)。

这里需要注意一点：在阶乘过程中去掉后缀0再继续计算对阶乘最后去掉后缀0之后的非零位结果是没有影响的。同理，去掉前面 K 位对 K-1 位之后是没有影响的(第 K 为可能会受到进位的影响)，因此我们可以每次阶乘前去掉计算得到的后缀0然后对 10^{10} 取模，输出时再对 10^9 取模即可。P.S:这里的数据规模是 1 \leq N \leq 100,0000 ，因此不会超时。这道题在

Last non-zero digits of a very large factorial
